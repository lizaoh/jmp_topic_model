Sometimes it is convenient to represent a partial order ( P ,⩽) by an isomorphic suborder of the power set 2 
                         Q 
                       of some set  Q , ordered by set inclusion. One such example is the principal ideal representation defined in  Definition 2 . It belongs to the class of basic representations (cf.  Definition 1 ) which are characterized by the property that each  p ∈ P  is represented by a set which contains at least one more element than the union of its predecessors. This concept stems from considerations in knowledge space theory (see  Doignon and Falmagne, 1999 ) and is investigated in  Suck (2003) . There it was shown that  parsimonious  set representations are of particular interest. They add exactly one element to the union of representing sets of the predecessors. Again the principal ideals constitute an obvious example. However, it is likewise easy to construct orders which possess more such representations.  Suck (2003)  asked the question which structural conditions characterize those orders which have only one parsimonious set representation, namely the principal ideals.

In the present paper this question is completely answered. Furthermore, a few properties are derived on the minimal cardinality of the set  Q  which allows such a parsimonious representation. The final section connects these results to knowledge spaces. This connection was the main motivation to undertake the present research.

In the rest of this introductory section we present the precise definitions of the concepts used above and recall a few basic properties.

The concepts of knowledge space theory are related to other mathematical notions. A knowledge space is easily shown to be a lattice and each (finite) lattice can be represented as a knowledge space. A quasi-ordinal space corresponds to a distributive lattice and a well-graded space to a join-distributive lattice. Moreover, the dual of a knowledge space, i.e., a system closed under intersection has been studied as ‘closure systems’.

The basis contains all the information of a knowledge space in a compressed form. It is unique. See  Doignon and Falmagne (1999, Chapter 1, paragraphs 1.19–1.34)  for more information and algorithmic aspects concerning the basis.

It is however not obvious how the properties of a knowledge space can be directly read off from the basis. This question seems to be relevant since the basis is usually much smaller than the space. It is therefore preferable to work with the basis whenever possible.

In particular the result on well-graded spaces is possibly of interest in the application of knowledge spaces. As pointed out by  Koppen (1998)  well-gradedness is a compelling property which is not easy to characterize in an empirically useful way.  Theorem 5  (and its proof) suggests how to find the well-graded spaces which have a given abstract order: Construct the principal ideal representation; then delete elements such that the resulting system is still a parsimonious set representation. How can one do this? For each collection of parallel fans which are not skewly topped one element can be deleted. Furthermore, this is the only way to reduce the number of elements used in the representation while keeping well-gradedness.

Clearly, the question is important for applications how to determine a partial order for a given field of interest which qualifies as an abstract order of the knowledge space to be constructed. Possibly, this problem can be solved by using the connection between skill structure and basis (cf.  Doignon and Falmagne, 1999 , Chapter 4,  Doignon, 1994 , or  Gediga and Düntsch, 2002 ). The skill structure can be obtained from experts of the field. This structure should be a partial order. Any parsimonious set representation of it gives rise to a well-graded knowledge space. The elements of the representation are the items of the test. Of course the feasibility of such a procedure depends on the (logical) structure of the field for which the test is constructed. Furthermore, theoretical results or algorithmic procedures seem to be called for to actually carry through the transition from the skills to the items of the test via parallel fans.
