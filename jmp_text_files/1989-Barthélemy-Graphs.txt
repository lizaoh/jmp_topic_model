INTRODUCTION  Partially labelled trees are used to model divisive, bifurcating, filiation, evolutionary, and similar processes, as well as  taxonomic structures (see Barth6lemy & Gu6noche, 1988). Since the method ADTREE was proposed by Sattath and Tversky (1977),  such trees have frequently been used to analyze metric or similarity data. From a nonmetric point of view, Buneman (1971)  describes them in terms of dichotomous data on a finite set and Colonius and Schulze (1981) describe them in terms of  quaternary relations.  Median graphs appear as a generalization of both trees and diagrams of distributive lattices (see Mulder, 1980a; Bandelt and  Hedlikova, 1983). In that respect they involve betweeness and order as well as dimensional notions and constitute a good  candidate to represent qualitative observations.  The aim of this paper is to show how median graphs and trees can be used to represent and analyze dichotomous data. A set X of  dichotomous attributes on a set X of stimuli induces a 0/1 array with stimuli as rows and attributes as columns. The epithet  "dichotomous" means that we intend to treat this array in such a way that the result of the treatment would be the same if the  0 and the 1 of a column have been exchanged. So a dichotomous attribute just induces a bipartition on X.  The technique we present is very close in spirit to component analysis, even though it is purely qualitative. This space is not intrinsic  because it is built from the data (for instance the number of the variables gives its dimensionality). In this space the  stimuli become a set of points and the variables become the coordinates of these points. Since this set of points is not more  readable than the original data set, one seeks for representations on very simple euclidean spaces, namely the axes. So, using  an optimization process, a decomposition into axes (the so-called principal components) is obtained and each stimulus is  represented in each of these components.  Here, we choose to represent a data set (stimuli or individuals described by dichotomous attributes) in a median graph. This  graph is also not intrinsic, and it is built from the data. In this graph the stimuli are located on some vertices (we will  call them the actual vertices), while the attributes are represented on the edges. Since this organization is not more  readable than the original data set, we seek for representations on very simple median graphs, namely on trees. Using an  optimization process a decomposition into trees is obtained and each stimulus is located at one vertex of each of these trees.  These analogies indicate the plan we shall follow in this paper. The first section is devoted to median graphs with latent  vertices (the basic structure to represent sets of dichotomous data). The second section establishes the way to represent a  set of dichotomous data in a median graph and studies the special case of trees. The third section describes the optimization  process that leads to the decomposition of the data set as a "sum" of trees. Combinatorial optimization is not as simple as  the optimization of quadratic forms, and this optimization process will be proved to be NP-hard. An application to a real data  set is briefly discussed in the last section.  1. Some Remarks  As we have just seen, in the X-median graph (G(L'), f,), actual vertices are the Helly locations of E and, in terms of  dichotomous data, they represent individuals (or stimuli). Latent vertices are locations which are not Helly. Moreover, every  vertex with degree at most two is an actual vertex (Lemma 1). So vertices with degree > 2 may be constructed from actual  vertices with the help of a sequence of ternary median operations. TREE ANALYSIS OF DICHOTOMOUS DATA  Tree Decompositions  We now know how to represent any set L of dichotomous data describing the set X as an X-median graph. As we mentioned in the  Introduction, this technique may be seen as somehow close in spirit to the classical representation of numerical data as  points in an euclidean space. But, like euclidean representations, the Xmedian graph (G(L), fs) may be somewhat complicated  and, instead of considering it, we could try to decompose L into simple structures as is done in component analysis when the  spatial representation of a set of numerical data is decomposed into "optimal" axes. But of course the reason to do that is  more serious than the difficulty of visualizing the global representation. For instance, in component analysis, the so-called  principal components (that is to say the optimal axes) allow one to detect "oppositions" between the represented individuals  (or stimuli) and to interpret them. Here we shall use a decomposition into trees that will permit us to detect common  filiations (a and b in the X-tree of Fig. 1), clusterings vs oppositions between groups of individuals (a, b vs c, d, e in  Fig. 1), and tendencies to be Guttman, as well as partial taxonomies and, when they are at the extremities of a "long" path,  oppositions between two individuals x and y (this means that there exist "many" dichotomies { E, E' } such that x e E and y e  E').
