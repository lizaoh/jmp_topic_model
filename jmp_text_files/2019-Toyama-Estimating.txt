Accumulating studies have supported the concept that value-guided choice is controlled by at least two systems of reinforcement learning (RL) ( Daw, Gershman, Seymour, Dayan, & Dolan, 2011; Daw, Niv, & Dayan, 2005; Dolan & Dayan, 2013 ). One is the model-free learning system, which is purely experience based and prompts the learner to repeat rewarded actions using reward-prediction errors. The other is the model-based learning system, which flexibly uses a mental model of an environment to calculate action values. The relative weighting of these two systems can be estimated by computational models or logistic regression models for two-stage decision tasks, a type of test first developed by  Daw et al. (2011) . The balance between these two systems reportedly varies depending on working-memory capacity ( Otto, Gershman, Markman, & Daw, 2013 ;  Otto, Raio, Chiang, Phelps, & Daw, 2013 ) and developmental changes ( Decker, Otto, Daw, & Hartley, 2016 ). An imbalance between these two systems, especially decreased model-based weight, is considered to be associated with several psychiatric disorders and constructs involving compulsivity, such as obsessive–compulsive disorder (OCD), eating disorders and alcohol dependence ( Gillan, Kosinski, Whelan, Phelps, & Daw, 2016; Patzelt, Kool, Millner, & Gershman, 2018; Voon et al., 2015 ).

The relative contributions of the model-free and model-based systems to choices can be inferred from a parameter value of the computational model developed by  Daw et al. (2011) . However, the validity of the computational model has not been sufficiently examined. If the model is insufficient or inappropriate, then the estimated parameter value contains some noise or biases. The possibility of these biases must be clarified considering that the two-stage decision task and its computational models are widely used in the fields of psychology, psychiatry, and neuroscience. In this study, we primarily focused on the assumption used in model-free value calculation and the assumption about action perseveration as factors that can bias the weighting parameter of the model-free and model-based systems, which are explained in detail later. These assumptions relate to any RL models. Thus, we believe that the biases examined in this study could provide insights into studies using RL models.

This study was inspired by our previous finding ( Toyama, Katahira, & Ohira, 2017 ) that the model-free system can explain choice data better using the model including the forgetting process, which assumes that learned values decay over time when the options are not chosen. This assumption has not been used in most previous models for the two-stage decision task, thus prompting the following question: Does any critical bias exist in the estimation of the weighting parameter of the model-free and model-based systems by fitting data using a model without the forgetting process? If so, the previous findings using this parameter must be reinterpreted.

In addition, we questioned whether the balance between the model-free and model-based systems is the most critical feature for characterizing a broad range of mental disorders and psychological traits. For example, to the best of our knowledge, in studies using two-stage decision tasks, OCD has been associated exclusively with an imbalance of the two systems ( Gillan & Daw, 2016; Voon et al., 2015 ), and the perseveration parameter values of OCD patients did not differ from those of nonclinical control participants ( Voon et al., 2015 ). Conversely, studies using the Wisconsin Card Sorting Test (WCST) have repeatedly shown that OCD patients have increased perseverative errors (e.g.,  Lucey et al., 1997; Okasha et al., 2000 ). Similarly, OCD is associated with stronger habitual responses in outcome devaluation tasks ( Gillan & Robbins, 2014 ). These findings show diminished behavioral flexibility in OCD patients, and this inflexibility is likely to cause repetitive compulsive behaviors in the form of rigid rituals, which characterize OCD. We must examine the possibility that the existing computational model for the two-stage decision task is insufficient to capture the unique feature of perseveration in OCD. Specifically, although the perseveration term of the existing computational model captures the repetition tendency of only the last action, there is the possibility that the actions executed before the last action can also affect the current choice.

If the computational model is insufficient to capture a meaningful feature such as the forgetting process or perseveration, then the data structure reflecting the feature is absorbed by other existing parameters and causes undesirable estimation biases regarding these parameters, which can yield misleading results. In this article, we propose RL-based computational models for the two-stage decision task that are less restricted than the conventionally used computational model. We (1) simulate the estimation bias that model misspecification may cause in the weighting parameter between the model-free and model-based systems; and (2) re-examine the relationship between the learning process and OCD using these models.

Generally, RL algorithms include two steps. In the first step, action values are updated based on outcomes; in the second step, choice probabilities for the next trial are calculated using these action values. In the second step, some models also integrate the effect of choice history related to perseveration. In this study, we consider models that are less restricted in both steps than the commonly used model. Regarding the first step, we introduce an assumption that unchosen action values are also updated along with the chosen action values. In commonly used RL models, the values of unchosen actions are assumed to remain unchanged. In contrast, early-learned values can be assumed to be lost over time ( Barraclough, Conroy, & Lee, 2004; Ito & Doya, 2009 ) as a natural consequence of memory decay (i.e., forgetting). Our previous paper, using the two-stage decision task, demonstrated that choice behavior can be better explained by a model that includes this forgetting process than by the original model, which lacks this process ( Toyama et al., 2017 ). However, models with forgetting have not been used as a standard in other studies using the two-stage decision task, and it remains unknown whether exclusion of the forgetting process can bias the estimations regarding the weighting parameter of the model-free and model-based systems.

Regarding the second step of RL models, we introduce into the choice probability function a more flexible assumption concerning perseveration than found in generally used RL models. It was reported that an RL model including a perseveration term had a remarkably improved fit compared with that of a model lacking this term ( Worthy, Pang, & Byrne, 2013 ). The commonly used perseveration term in the two-stage decision task captures whether a previous action tends to be repeated (i.e., immediate perseveration). However, behavior cannot be explained solely by this assumption of immediate perseveration ( Akaishi, Umeda, Nagase, & Sakai, 2014; Lau & Glimcher, 2005 ). It is safe to assume that the current action exhibits autocorrelations with succeeding actions, with this correlation decaying gradually as trials progress (i.e., gradual perseveration). A possibility exists that the computational models used in previous studies were insufficient to capture action perseveration, which is one main symptom of OCD patients. In addition,  Katahira (2018)  reported that RL models with an immediate-perseveration term have strong bias in estimating learning rates, compared with models containing a gradual-perseveration term. In the current article, we conduct several simulations to examine the possibility that models with an immediate-perseveration term distort the estimated balance between model-free and model-based systems.

Thus, we introduce models that include the forgetting process and/or gradual perseveration. Standard RL models can be expressed as special cases of these models. Using these models, we examine the possible biases that can affect estimates of the weighting between the model-free and model-based systems. From our simulations, we find that there are systematic biases when the forgetting process, the gradual-perseveration process, or both are omitted from a fitting model in cases in which the true data-generating model includes them.

In addition, we test the possibility that computational models with a forgetting process and/or gradual perseveration can provide new insights into or conclusions about the relationship between OCD and learning processes. Using data obtained from 42 healthy individuals, we examine the relationships between the parameters estimated by several computational models and self-reported OCD severity scores. Applying a forgetting process and gradual perseveration to the fitting models, we demonstrate the possibility that individuals with higher obsessive–compulsive tendencies have both lower learning rates and lower forgetting rates than those with lower obsessive–compulsive tendencies.

In this article, we used the two-step decision task developed by  Daw et al. (2011)  to generate synthetic behavioral data and to obtain experimental data. In the simulation, the task included 303 trials; in the experiment, the task included 240 trials. Each trial had two stages ( Fig. 1 A). The first stage (state A) used two stimuli consisting of fractal images. Agents chose one stimulus (with a mouse click in the experiment), and the second stage began with the presentation of either state B or state C. One stimulus led to state B at a rate of 70% (common transition) and state C at a rate of 30% (rare transition), whereas these probabilities were reversed for the other stimulus. This transition probability was fixed during the task. Agents again chose one of two stimuli in the second stage. Depending on their choice, feedback was displayed for 1 s to indicate whether the agents were rewarded or not rewarded. In both stages, if no response was made within 2 s, a message that said, “Too late!!” was presented, and the participants proceeded to the next trial. The reward probability of each second-stage stimulus was previously determined and was changed independently and slowly over the trials in a drifting Gaussian random walk ( SD  
                         
                            = 
                          0.025) between 0.25 and 0.75. In the simulation, the reward condition setting from our previous research ( Toyama et al., 2017 ) was applied to the entire population; in the experiment, the reward condition setting was different from that in the simulation but consistent across all subjects (see  Fig. 1 B). The purpose of this uniformity was to guarantee that any differences in choice results were not caused by different reward settings. In the experiment, the participants were told in advance that each first-stage stimulus had a different constant transition probability than the second-stage states and that the reward probabilities of the second-stage stimuli would change slowly and independently over time. The participants completed a 10-trial training session before the task. After this task, they also participated in other experiments irrelevant to the purpose of this article, and each subject received a total of 2200 yen as a reward for participation.

Fig. 1 C shows a prediction of the first-stage choice under each condition of the previous trial by purely model-free and purely model-based agents. A model-free agent repeats the choice of a rewarded stimulus and avoids choosing an unrewarded stimulus. In contrast, a model-based agent uses information about transition probabilities to determine which of the first-stage stimuli is more likely to lead to the second-stage state in which the agent was rewarded or not rewarded in the previous trial. With a model-based agent, the interaction between the experienced transition and reward outcome is assumed.

In the experiment, after the behavioral tasks, the participants answered several questionnaires. The questionnaire relating to the purpose of this article was a self-report scale of OCD symptoms. For the analyses of the experimental data from the two-stage decision task, we excluded uncompleted trials, i.e., those in which a choice was not made within 2 s. Trials were also excluded if the response latency was too short (less than 120 ms) because we deemed the participants unlikely to have considered the stimulus types in such a short time. We excluded 2 participants who missed more than 10% of the trials according to the above criteria. We also excluded 1 participant who chose the same first-stage action in more than 90% of the trials.

We first introduce two computational models for the two-stage decision task that have exactly the same model-free algorithm but different model-based algorithms: an original model developed by  Daw et al. (2011)  and the model proposed in our previous paper, which showed a better fit than the former ( Toyama et al., 2017 ). We refer to the former model as the parallel model. In this model, the model-free and model-based values are assumed to be calculated in parallel. We refer to the latter model as the eligibility-adjustment (EA) model. In this model, the degree of value updating using the eligibility trace is determined by the balance of the model-free and model-based systems. The detailed differences between these two models are not the subject of the current article but have been discussed in  Toyama et al. (2017) .

Subsequently, we introduce two assumptions that can be combined with the parallel and EA models. These assumptions are the main objects of examination in this paper: the forgetting process, which assumes that the unchosen option values are updated; and perseveration, which assumes that actions have autocorrelations with the preceding actions.

Next, we conducted model-neutral analyses of the experimental data to test for the existence of the forgetting process and gradual perseveration, and our findings supported their existence (see Supplementary Text 1, Supplementary Text 2 and Table S1). However, most previous studies using the two-stage decision tasks did not use models with forgetting or gradual perseveration, and it remains unknown whether the omission of these processes could lead to serious estimation bias in the weighting of model-free and model-based systems. In Section  5.2 , we simulate what occurs when fitting models without forgetting are used on data generated by models with forgetting, after which we simulate what occurs when fitting models with immediate perseveration are used on data generated by models with gradual perseveration. In Section  5.3 , using the experimental data, we compare the computational models with different assumptions and examine the relationships between the estimated parameters of these models and obsessive–compulsive tendencies.

We revealed that the estimation of the weighting between the model-free and model-based systems can easily be distorted by model misspecifications. This article showed two conditions under which such estimation biases can occur. First, when choices were made by an agent with forgetting but the data were fitted by a computational model without forgetting, the weighting of the two systems was systematically biased in the model-free or model-based direction. Second, when choices were made by an agent with a gradual-perseveration tendency, but the data were fitted by a computational model expressing only the immediate-perseveration effect, the weighting of the two systems was always biased in the model-free direction. In addition, our experimental data demonstrated that individuals with higher obsessive–compulsive tendencies have both lower learning rates and lower forgetting rates than those with lower obsessive–compulsive tendencies.

Our simulations revealed several possible estimation biases due to model misspecification. In computational models that include multiple parameters and that assume their complex interactions, it is sometimes difficult to completely explain the

mechanism whereby one parameter value affects another. Nevertheless, the structure of the models suggested two plausible mechanisms that could cause the biases in parameter  
                            w 
                          seen in our simulations. First, the biases seen, especially in parameter estimation by the parallel model without forgetting, occurred because the weighting parameter w reflects not only the weighting of the two systems but also the degree of misspecification of these systems in a computational model. In general, when the model construction of one system is insufficient to capture data, the weighting of this system regarding choices will be devaluated. In the parallel model, the model-free values are calculated mainly using the first-stage action values, and model-based values are calculated using the second-stage action values. Consequently,  
                            w 
                          is biased in the model-based direction when first-stage updating of the unchosen options is described inaccurately (i.e., without forgetting), while the variable is biased in the model-free direction when second-stage updating of the unchosen options is described inaccurately. Thus, the parameter  
                            w 
                          has a bias due to the difference in modeling accuracy between the two systems.

These types of biases occur because the weighting parameter  
                            w 
                          represents the relative contributions of the two systems. Thus, regardless of whether the model misspecification is related to the model-free system or the model-based system, the model misspecification always leads to estimation bias of this parameter. However, the model used by  Otto, Raio et al. (2013) , which is simply a reparameterization of the parallel model, provides the weighting parameters separately for the model-free and model-based systems (see Supplementary Text 3). In some cases, this model enables the restriction of the estimation biases to one system weight without any influence on the other system weight. However, in other cases, the estimation biases of one system can affect the other system weight because the model-free and model-based values are not completely independent (see Supplementary Text 3 and Figure S2). Thus, when modeling multiple systems that are not completely independent, the weighting parameters should be carefully interpreted because there is the possibility of undesirable estimation biases.

Estimation biases also occur when a portion of the data can be predicted not only by a parameter related to their generation but also by one or more parameters irrelevant to their generation. In this case, if a truly relevant parameter is excluded, the other parameter adjusts its value to improve the model fit to the data. The resulting fit is undesirable and causes estimation bias. In our simulation, such a case occurs when first-stage forgetting is not included in the EA model and when the immediate-perseveration models are used for data produced by the gradual-perseveration models. Critically, the tendency toward action repetition can be expressed by several parameters in a model. The most direct parameters are those of the value-independent part (perseveration strength  
                            ϕ 
                          and its decay rate  
                            τ 
                         ). In addition, both weighting parameter  
                            w 
                          and the forgetting rate  
                            
                               
                                  α 
                               
                               
                                  F 
                               
                            
                          relate to action-repetition tendencies. In particular, the model-free system (lower  
                            w 
                         ), by its nature, generates data including more action perseveration than the model-based system (higher  
                            w 
                         ) in the two-stage decision task. Consequently, when the fitting model does not include forgetting and gradual-perseveration processes, the parameter  
                            w 
                          is inevitably biased to complement the action-repetition tendency actually caused by these processes.

These types of biases are not specific to the model of the two-stage decision task. Similar bias can occur for any model including a weighting parameter of plural systems or any model including multiple parameters that relate to the perseveration tendency. In addition, the same mechanism discussed above can explain some of the estimation differences in other parameters due to different model constructions. The values were consistently smaller in the models with the gradual-perseveration term than in the models with the immediate-perseveration term when the other model assumptions were the same. This difference might have occurred, at least in part, because a portion of the perseveration tendency expressed by the value calculation in the immediate-perseveration models was expressed by the value-independent perseveration term when the gradual-perseveration models were used.

Regarding the model assumption about the updating of the unchosen options, we used the models with forgetting as a less constrained option than previously used models, which assumed that unchosen option values remain unchanged. We have shown that the computational models with forgetting were supported by model selection, in accordance with our previous report ( Toyama et al., 2017 ). A similar approach with forgetting is known as the decay learning rule ( Erev & Roth, 1998 ). This rule is similar to our model in which the default value is zero.  Worthy et al. (2013)  noted that the decay rule tends to prompt the perseveration tendency, as also discussed in our article on the case of models with forgetting and a default value of 0. Using less constrained models, we also explained why less perseveration can be seen when the default value is high.

Regarding the assumption of the value-independent perseveration effect on choices, we used the models with the gradual-perseveration term as a less constrained option than those with the immediate-perseveration term, which have often been used in previous studies. The models with the gradual-perseveration term were not supported by model selection. However, statistical analyses of experimental data revealed that participants tended to repeat actions not only from the immediately preceding trial but also from earlier trials. In addition, for a few participants, large estimation biases for  
                            w 
                          were seen when the immediate-perseveration term was used instead of the gradual-perseveration term. In previous research using the Iowa gambling task, it was reported that the association between task performance and parameter estimation was strengthened when the value-expectation term was decomposed from the perseveration term ( Worthy et al., 2013 ). The same phenomenon occurred in our results. The relationships between the learning rate and obsessive–compulsive tendencies and between the forgetting rate and obsessive–compulsive tendencies were clearer when the models with a gradual-perseveration term were used than when the models with an immediate-perseveration term were used. The likely reason is that the gradual-perseveration term reduced the effect of the value-independent choice tendency on the estimation of parameters relating to the value-calculation process. With these considerations in mind, we recommend the inclusion of the gradual-perseveration term in the RL model to improve the parameter estimations and to prevent possible estimation biases.

In most previous studies, these assumptions were not included in the computational models, implying that the weighting parameter  
                            w 
                          reported in these studies might have had some bias. If such a bias always occurs in the same direction and with the same magnitude for everyone, the bias will not be a serious issue in studies of relative comparisons. However, our simulations show that this case does not apply. Our simulation showed that the direction and magnitude of biases could vary depending on the missing true parameter values of individuals. Consider the situation in which a researcher is interested in the comparison of the model-based weights of two groups and uses a model without forgetting to evaluate the weights. In this case, if one group actually has a strong forgetting effect on the data while the other does not, proper comparison of the model-based weights is impossible because of the unequal biases between the groups. Thus, it is better to use the model that can prevent potential estimation biases. Fortunately, widely used standard RL models are nested within the RL model with forgetting and gradual perseveration assumptions. It is safe to use the model with these assumptions to avoid possible biases in future studies.

Our results suggest the possibility that typical OCD symptoms, such as repetitive compulsive rituals, result from the small magnitude of value updates. In our experiment, individuals with high OCI scores showed relatively sustained values of both chosen and unchosen actions despite new experiences. Importantly, this pattern was not found until the computational models with forgetting were used. Our result is not consistent with the finding by  Vaghi et al. (2017)  of high learning rates in OCD for spatial action adjustment from feedback. In contrast, our result of a low learning rate in individuals with high OCI scores is consistent with previous studies using several decision-making tasks. It has been reported that OCD patients gathered more evidence than healthy controls in the beads task ( Fear & Healy, 1997; Péllisier & O’Connor, 2002 ). In the perceptual discrimination task using random dot motion, OCD patients showed slower evidence accumulation than controls under high-uncertainty perceptual contexts ( Banca et al., 2015 ). OCD patients are also insensitive to outcome devaluation ( Banca et al., 2015; Gillan & Robbins, 2014 ). These previous findings might be related to slow value updating. However, further research is needed to draw conclusions regarding the relationship between OCD and learning rate, which could be affected by several factors, such as differences in action type, the computational model used, cognitive load or uncertainty in the task.

We did not find any relationship between the perseveration parameters ( 
                            ϕ 
                         ,  
                            τ 
                         ) and obsessive–compulsive tendencies. This result is in accordance with a previous study using the same task and computational model with the immediate-perseveration term ( Voon et al., 2015 ). It might be concluded that there is little or no relationship between the perseveration tendency captured in the two-stage decision task and the obsessive symptoms of OCD. It has been reported that, in other decision-making tasks, OCD patients show more ambiguity aversion and are less consistent in their choices than healthy individuals ( Hauser et al., 2017; Pushkarskaya et al., 2015 ). These findings imply that OCD patients might not show high perseveration in choice tasks among high-uncertainty options. Conversely, the perseveration parameterized in the two-stage decision task is probably distinguished from the increased perseveration in OCD on the WCST and the devaluation tasks ( Gillan & Robbins, 2014; Lucey et al., 1997; Okasha et al., 2000 ). The former is value-independent perseveration during learning, whereas the latter is perseveration after the action is connected to positive outcomes.

The current results did not support the previously reported association of  
                            w 
                          with OCD patients ( Voon et al., 2015 ) or with OCD tendency in large nonclinical populations ( Gillan & Daw, 2016; Patzelt et al., 2018 ); however, our correlation analyses still showed a negative correlation with the weighting parameter  
                            w 
                          (although the correlation was not significant). Considering that the participants in our experiment were not clinical patients, and the sample size was small, further examinations are necessary to draw conclusions regarding the relationship between model-based weight and OCD. For example, web-based data collection is a useful approach to obtain the sheer size of a nonclinical sample and to examine the data from participants with various symptom severities ( Gillan & Daw, 2016; Gillan et al., 2016; Patzelt et al., 2018 ). The significant relationship between the model-based weight and OCD might be first seen in such a broader population with a wide variety of severities. Conversely, our current results suggest another possibility: that abnormality of the learning rate or the forgetting rate leads either directly or indirectly to the apparent imbalance between the model-free and model-based systems. Further research is needed to clarify which processes of decision making most characterize OCD patients.

In this study, we demonstrated the possible biases in parameter estimation and examined and discussed their mechanisms. Before the conclusion, we present some points to reduce these biases and to avoid incorrect conclusions about the estimated parameters.

Task design

• 
                               The goodness of a computational model is limited by the task design from which the data are obtained. To reduce the noise in parameter estimation, it is important to suppress the possible strategies that participants might use in the task beyond the experimenter’s assumptions.

As demonstrated in our study, if a parameter responsible for a particular choice tendency is not included in a fitting model, other parameters might show a pseudo-fit to this choice tendency.

If bias is caused by a model fixing some parameter value, this parameter should be included in a model even if the model comparison supports this reduced model.

Evaluate the parameter estimation accuracy. For this evaluation, simulations are useful, confirming that the model can produce the same parameter values with preset true parameter values. (This topic is discussed in detail by  Palminteri, Wyart, and Koechlin (2017) .)

The parameters of computational models can have undesirable biases due to model misspecifications. In this article, we used the two-stage decision task to demonstrate the estimation biases concerning the balance between the model-free and model-based learning systems. Based on our results, we recommend the use of models with the forgetting-process and gradual-perseveration terms to prevent these biases. In addition, these models provided the new finding that individuals with higher obsessive–compulsive tendencies have both lower learning rates and lower forgetting rates.

The estimation biases and related mechanisms discussed here could be helpful in constructing new computational models and interpreting the parameters of existing models. In addition, because a computational model is a combination of hypotheses, it is necessary to examine the validity of the hypothesized processes in a model. Such examinations will increase the utility of computational models in understanding the cognitive and neurobiological processes underlying behavior and the symptoms of psychiatric illness.
