JOURNAL OF MATHEMATICAL PSYCHOLOGY 23, 257-272 (1981)  Joint Independent Random Utility Models Where One of the Choice Structures Satisfies the Strict Utility Model  A. A. J. MARLEY,  Department of Psychology, McGill University  It is known that one cannot construct an independent joint random utility model for which both the preference and aversion  probabilities satisfy the strict utility model. Given this result, it is natural to ask what kind of relations are possible in  such contexts. Here, for instance, we demonstrate that if a structure of preference and aversion probabilities are generated  by a common independent random utility model, and if the preference probabilities satisfy the strict utility model, then the  preference and aversion probabilities are related by what is known as the strong acceptance condition. We also prove an  appropriate converse of this result. Finally, an example is given of a process (other than the random utility model) that is  compatible with the assumption that a structure of preference and aversion probabilities satisfies a common strict utility  model.  The strict utility model (or, as it is usually known, Luce's choice model) has been extensively studied both theoretically and  empirically (for a recent summary, see Luce, 1977). There are numerous examples of its satisfactory description of data, and  in those cases where it fails, various generalizations of the basic idea seem viable (e.g., Tversky, 1972a, 1972b). It is  therefore valuable to have as complete an understanding as possible of the properties of this model, not only because of the  model's own applications, but because results obtained for the choice model often generalize in natural ways to broader  classes of models. In this paper, we study relations among the strict utility model, the strong acceptance condition, and the  strong rejection condition (both of the latter discussed in Marley, 1968), this study taking place in the context of the joint  random utility model (Marley, 1965). DISCUSSION  Given the initial result that we cannot have a joint independent random utility model in which both the preference and the  aversion probabilities satisfy the strict utility model, we can ask if there is any "natural" process that is compatible with  this latter constraint. We now construct such a process.  Assume that each x E T has associated with it a class of desirable features, denoted by "1," and a class of undesirable  features, denoted by "0." When inspecting an object for choice, the subject selects a feature at random, which thus may be  desirable or undesirable. Let Ux denote the feature distribution, and let u(x) = Pr(U, = 1), and so 1 - u(x) = Pr(Ux = 0). To  end up with the strict utility representation, we must restrict attention to elements x E T with u(x) * 0, 1. Finally, when a  subject is presented with a set of objects A from which he must choose the "best" (preference), he inspects each element of A  in turn, and if a single element has a desirable feature selected, then that element is chosen; otherwise the subject  resamples on the whole set.
