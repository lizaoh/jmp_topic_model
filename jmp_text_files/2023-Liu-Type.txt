Experimental psychologists constantly seek to determine whether a behavior is regulated by a single or by multiple latent variables. The classic approach is functional dissociation indicated by the interactions of analysis of variance (ANOVA). The logic of dissociation, however, is arguably problematic and fails to yield a concrete conclusion for multiple latent variables, because a single latent variable can possibly affect two tasks with differen t  strengths or directions to produce a significant interaction effect ( Davies, 2010; Dunn & Kirsner, 1988, 2003 ). Therefore, some researchers have proposed that state-trace analysis (STA;  Bamber, 1979 ) offers a superior methodology for evaluating the unobserved structure ( Dunn & Kalish, 2018; Dunn et al., 2012; Newell & Dunn, 2008; Stephens et al., 2019, 2020 ).

One quantitative analytic technique for STA is based on conjoint monotonic regression (STA-CMR;  Dunn & Kalish, 2018; Kalish et al., 2016 ). Since both ANOVA and STA-CMR adopt null hypothesis significance testing (NHST) for statistical inference and their results are comparable in a three-factor design with linearity assumption ( Dunn & Kalish, 2018 ), it is of interest to compare type I error and statistical power between them.

This article starts with a brief overview of STA logic and then moves on to an introduction to the design and quantitative analysis of STA. Next, using simulated data from a three-factor experiment design, the type I error and statistical power of STA-CMR and ANOVA are examined in both linear and non-linear models, and the reason for the smaller-than-expected type I error rate of STA-CMR is explored. Finally, the pros and cons of STA-CMR are discussed based on the simulation results.

State-trace analysis is a methodology for exploring the number of latent variables that mediate the effects of independent variables on dependent variables. Specifically, STA can be used to discriminate a model with single latent variable from a model with multiple latent variables through a scatter plot called state trace plot, which illustrates the performance on two dependent variables against each other.

There are two kinds of functions in STA. The mappings of independent variables to each latent variable are called input functions and the mappings of latent variables to each dependent variable are called output functions. According to the logic of STA, when there is only one latent variable, the state trace plot is one-dimensional, that is, a monotonic or non-monotonic curve. Specifically, when each output function is monotonic, they must have inverse functions. One of the inverse functions can be nested into the other monotonic output function, making the state trace plot form a monotonic curve ( Fig. 1 A). On the other hand, when one of the output functions is non-monotonic, the state trace plot forms a non-monotonic curve ( Fig. 1 B). And we should pay attention to a special case that non-monotonic output function with discontinuity points (e.g., tangent function) since it has the potential to produce multiple curves in a limited range that resemble a multi-dimensional state trace plot ( Fig. 1 C). As for multiple latent variables, the state trace plot may be one-dimensional or two-dimensional. The former appears if the input functions ( Fig. 1 D) or output functions ( Fig. 1 E) are functionally dependent ( Dunn & Kalish, 2018 ), and the latter shows up when there is no functional dependence ( Fig. 1 F).

In practice, we would like to infer the number of latent variables from the dimension of the state trace plot. However, as can be seen from the logic above, it is challenging to put into practice. This is because if a one-dimensional state trace plot appears, then there may be only one latent variable, or multiple latent variables with functionally dependent output/input mappings. And if a two-dimensional state trace plot is found (e.g., multiple curves), then there may be a single latent variable with a non-monotonic output mapping, or multiple latent variables without functional dependence. Furthermore, because an observed state trace consists of a finite number of points, which represent the combinations of independent variables, distinguishing a single non-monotonic curve from a two-dimension area is difficult ( Ashby & Bamber, 2022; Dunn & Kalish, 2018 ). Therefore, in order to make STA more practical, an additional constraint is required, that each output function from the single latent variable to the dependent variables is monotonic ( Benjamin et al., 2019; Kalish et al., 2016; Loftus et al., 2004; Prince et al., 2012 ).

According to the monotonicity assumption, if all the points on the plot can be fitted by a single monotonic curve, there may be either a single latent variable or multiple latent variables with functionally dependent mappings. Notably, if the points cannot be fitted by a one-dimension monotonic curve, it can be inferred that the state trace plot is a two-dimension area, suggesting the existence of multiple latent variables ( Ashby & Bamber, 2022; Dunn & Kalish, 2018 ). Based on this assumption, several quantitative statistical methods for STA have been developed and applied in several fields of psychology research, such as face recognition ( Loftus et al., 2004; Prince & Heathcote, 2009, 2010 ), associative recognition ( Hamm, 2014 ), and category learning ( Dunn et al., 2012; Stephens et al., 2019 ).

Although the monotonicity assumption is not required for classic STA, it improves the applicability and makes STA inference more straightforward ( Bamber, 2019; Dunn & Kalish, 2018 ). However, some researchers argued that this assumption may be violated in many single latent variable models ( Ashby, 2014; Ashby & Bamber, 2022 ). There are indeed many non-monotonic output mappings, but there are also a large number of output mappings that can be logically considered to be monotonic in psychological research ( Dunn & Kalish, 2018; Dunn et al., 2014; Wagenmakers et al., 2012 ), such as the relationship between recognition accuracy and strength of memory ( Dunn et al., 2014 ). Besides, if we discard the monotonicity assumption because it is too strong to be justified, all ANOVA results should also be dismissed because the linearity assumption is an even more constrained version of monotonicity. It should be noted that monotonicity is just an assumption, and we will only employ it when it can be reasonably assumed. If there is sufficient evidence to infer a non-monotonic output mapping, we shall consider using a different approach for our data.

It has always been debatable whether STA can be used to determine the number of underlying processes or systems. Some researchers argue that this question is beyond the scope of STA ( Ashby, 2014, 2019; Ashby & Bamber, 2022 ), while others believe that the number of latent variables places a lower bound of the number of processes or systems ( Dunn & Kalish, 2018; Dunn et al., 2014; Stephens et al., 2020 ). This debate will continue until the term “process” or “system” in psychology is clearly defined. Regardless of their disagreement, they both concur that STA can be used to determine the number of latent variables or parameters. As a result, we will refer to latent variable instead of process or system in this article.

Practically, because more points on a state trace plot can improve the diagnosis of monotonicity, at least one factor requires a sufficient number of levels in a two-factor design for STA. However, it is challenging to set so many levels for either factor in some experiments. For example, in a 2 (gender of participants)  
                         × 
                       2 (gender of faces to be recognized) design to investigate the own-gender bias in face recognition ( Herlitz & Lovén, 2013 ), it is obviously difficult to add more levels for either factor. An experiment design proposed by  Prince et al. (2012)  can be an appropriate solution for dealing with this issue. They suggested using an additional trace factor, which could necessarily generate more points on the plot. However, it should be noted that the trace factor should be chosen very carefully according to three criteria. Firstly, the trace factor must have a monotonic effect within each condition. Secondly, there should be an enough overlap between the traces of each condition. Lastly, the interactions between the trace factor and other factors should be taken into consideration ( Prince et al., 2012 ). Following this suggestion, the experiment becomes a three-factor design ( Fig. 2 ).

Errors, however, like all the other statistical methods, cannot be eliminated in STA, so mere observation of the forms of the state trace plot is insufficient to confirm whether or not the points fall on a single monotonic curve. Thus, certain techniques are needed to deal with it. One of the quantitative analyses for STA is conjoint monotonic regression (STA-CMR), proposed by  Kalish et al. (2016)  using a bootstrap method. The logic of STA-CMR is to compare the goodness-of-fit between a partial order model where a subset of points on state trace plot has the same specific order (partial order) on the  
                         x 
                      - and  
                         y 
                      -coordinates, and a monotonic model with an additional constraint that the  
                         x 
                      -coordinates have exactly the same order as  
                         y 
                      -coordinates for all points. Given that the monotonic model is nested within the partial order model, if the monotonic model is true, the partial order model is also true, resulting in a small difference in fits (DIF, for short) between the two models. As a result, if we calculate a large number of DIFs using bootstrap samples and check how many of them are larger than the original one based on raw data, the proportion will be high. But if we find that the proportion is less than the criterion, for example,  .05 , then the null hypothesis that the monotonic model is true can be rejected. The STA-CMR expects a monotonic increasing relationship between the dependent variables by default. This does not, however, imply that STA-CMR can only be used with state traces that are monotonically increasing, as we can reverse-code one of the dependent variables to make the relationship increase when the state trace appears to be decreasing.

It is important to highlight that the partial order is determined by prior knowledge about an expected order of the population means over a subset of conditions. If we have no prior knowledge about the partial order, it could be empty. In this case, the fit of partial order model must equal zero, hence the difference in fits between the two models turns to the fit of monotonic model itself. In other words, the partial order is optional. Besides, we can test the feasibility of the preset partial order by using a similar procedure to compare the fit between the partial order model and the model without partial order (unconstrained model;  Kalish et al., 2016 ).

NHST is adopted by the STA-CMR procedure for statistical inference. Further discussion of type I error and statistical power is therefore required. Although these two essential points have been discussed with simulated data ( Kalish et al., 2016 ), a direct comparison of these features between STA-CMR and ANOVA has not yet been performed. Given that in a three-factor design the interactions of ANOVA can result in two-dimension area on the state trace plot ( Dunn & Kalish, 2018 ), it means that the results from the two techniques are directly comparable.

The type I error is the probability of wrongly rejecting the null hypothesis, and is primarily determined by the preset significant level. The statistical power is the probability of correctly rejecting the null hypothesis, which is mainly affected by sample size and the ratio of variances between effect and error rather than the overall mean and the specific value of the effect. Thus we can cover a wide range of situations by changing the sample size and error variance while setting the overall mean and effects to fixed values.

Both in the linear and nonlinear models, the type I error rate of STA was less than the predetermined alpha. We considered three possible reasons for this result. First, the proportion of times that DIFs calculated from bootstrap samples (DIFs-B) exceeded that calculated from the observed data (DIF-O) may be greater for STA-CMR with a specified partial order than without a partial order. This is because the DIF-O of the former should in general be smaller than that of the latter. In other words, it may be more difficult to reject the null hypothesis for STA-CMR with the partial order constraint. Therefore, imposing a partial order may cause a reduction in the type I error rate. In addition, in STA-CMR with a partial order constraint, if the preset partial order is rejected due to unjustified prior knowledge or random errors, the test of the single latent variable model is uninterpretable ( Dunn & Kalish, 2018 ). Although the partial order in our simulations was determined by the preset parameters so the ‘prior knowledge’ should be justified, the partial order model still might be rejected in a small number of datasets because of random errors. Therefore, the estimated type I error rate might be inaccurate. Furthermore, the tolerance to random errors of STA may be another potential possibility. Theoretically, ANOVA assumes that the state trace plot will be fitted by a straight line of unit slope when there is no any interaction. But the random errors will definitely deviate the points from the straight line to a certain degree. From another point of view, the logic of ANOVA is to determine whether the degree of deviation is enough to infer an interaction. However, deviation from a straight line does not necessarily violate monotonicity, which is the null hypothesis of STA. In other words, it is much easier for the random errors to deviate the points from a straight line than to violate monotonicity ( Fig. 4 ). In the case where the monotonicity is unaffected by random errors, there is no need to perform STA as it will be impossible to reject the null hypothesis. Therefore, including the datasets which do not actually need to be analyzed may be the reason for suppressed type I error rate.

In order to test this, we performed a new simulation with the same parameters, but the data were analyzed by bootstrap ANOVA with only 100 bootstrap samples and by STA-CMR with 100,000 bootstrap samples. The results of bootstrap ANOVA showed that the estimated type I error rate was  .064 , slightly overestimated, and the mean and standard deviation of critical F were  3.707  and  0.698  respectively. Although the mean did not change much, the standard deviation increased nearly 10 times, resulting in a similar pattern to STA-CMR that some of the simulated datasets with larger F statistics were not rejected while some others with smaller F were rejected. On the other hand, the STA-CMR results showed that the estimated type I error rate was  .009 , and the mean and standard deviation of critical DIF were  2.466  and  0.315  respectively, very close to the previous simulation, indicating that the number of bootstrap samples might not be the reason for the small type I error rate.

Since the means of estimated critical F in the bootstrap ANOVA were very close to the true critical F, and the means of critical DIF were relatively unchanged in these two STA-CMR simulations, we tried to use the means of critical DIF as the true critical DIF to re-estimate type I error rate by calculating the number of simulated datasets whose DIF-Os were larger than the mean critical DIF. The results showed that the type I error rate were  .024  and  .022  for the first and second STA-CMR simulations respectively, slightly increased but still far from  .05 . This reminded us that the suppressed type I error rate was probably because the CMR bootstrap procedure overestimated the critical DIF. However, the reason is still unclear and needs further investigation.

STA is a statistical method for detecting multiple latent variables. If the linearity assumption is met, its inference of rejecting the single latent variable is equal to the inference of significant interactions of ANOVA. Our simulations showed that although STA-CMR control type I error rate much more strictly than ANOVA in linear model, its statistical power is lower. This is reasonable because all the assumptions of ANOVA are satisfied. Thus, ANOVA is more appropriate for data analysis in this case, resulting in higher statistical power. On the contrary, in the nonlinear models where the assumption of linearity is violated, ANOVA completely fails, whereas STA-CMR performs as well as in the linear models. This is because STA-CMR only assumes the monotonic output function, which is less strict than linearity.

Besides, although the partial order is optional for STA-CMR, imposing an appropriate partial order can increase statistical power in both linear and nonlinear models. However, an inappropriate partial order probably reduces the statistical power. Therefore, if we have some prior knowledge about the potential means of experiment conditions and want to set a partial order to increase the power, a test of the partial order model is essential before running the STA-CMR.

Furthermore, the most important finding of our simulations is that the estimated type I error rate of STA-CMR is lower than the preset significance level for both linear and nonlinear models. We tried to attribute the small type I error rate to the partial order constraint but found that neither removing the partial order constraint nor excluding the simulated datasets where the partial order model was rejected made the type I error rate approach significance level. In terms of the former, although imposing a partial order constraint does decrease the DIF-O, the DIFs-B will also be suppressed, so the type I error rate will not increase dramatically to approach significance level. As for the latter, since the partial order was justified and rejecting the partial order model rarely happened in our simulations, the type I error rate was essentially unchanged. We then tried to exclude the simulated datasets where the monotonicity were not affected by random errors to see if this could increase type I error to  .05 . This hypothesis, however, is also unsupported. Although none of the three hypotheses is supported, we find that the DIF distribution produced by STA-CMR bootstrap procedure is relatively unstable and the critical DIF is probably overestimated. We believe that this is strongly associated with suppressed type I error rate, but the cause needs further investigation.

Irrespective of the reason for suppressed type I error rate, these results indicate the rigorous control of type I error for STA-CMR, but it would also bring the disadvantage of reducing statistical power. According to these simulations, using a slightly higher significance level, such as

The true type I error rate of ANOVA in the linear models, on the other hand, was much higher than significance level. This inflation of type I error is identical to that in multiple comparison. The more mutually independent tests are conducted, the higher type I error rate will be. From this point of view, in a three-factor design with the linearity assumption acceptable, STA can be regarded as an overall test for all possible interactions between task factor and any others.

There are some limitations of this study. First, we only focused on a three-factor design. However, a two-factor design in which one of the factors has more than two levels could also be analyzed by it. Second, we only took the interaction between A and B into consideration for convenience. So, we cannot know if the statistical power of STA-CMR will change when additional interaction terms are put into the linear model. Next, we only used a power function and an exponential function as output functions in the nonlinear models. As a result, the impact of other types of monotonic output functions on the outcomes still needs to be specified. Finally, the reason for the suppressed type I error rate of STA-CMR is still unclear. After all, this is only a preliminary study, and all of the limitations necessitate further investigation.

In conclusion, the statistical power of STA-CMR is lower than ANOVA in the linear models, but confirmation of linearity assumption is difficult in practice. Given that STA-CMR can also be applied in nonlinear models, it could be a better choice when we aim to explore the number of latent variables under certain behavior.
