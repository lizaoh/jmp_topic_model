Purdue University, West Lafayette, Indiana 47907  A problem which arises in the analysis of reaction times is considered. Suppose a task requires the completion of a set o mental activities which can be represented as a partially ordered set of arcs in a critical path network, but the network is  unknown. If one knows for every pair of activities whether the pair is comparable (i.e., sequential) or incomparable (i.e.,  concurrent), then a partial order for the activities can be constructed with a procedure known as the transitive orientation  algorithm. It is known that only two partial orders are possible, one the converse of the other, unless some proper  nonsingleton partitive subset of the activities is not stable. Two main results are presented: First, a set of arcs generating  a weakly connected subnetwork is partitive if and only if the subnetwork has a unique source, a unique sink, and no vertex of  attachment other than these. Second, a simple relationship between partitive sets of activities and the slack between  comparable activities is presented. These issues are important for the uniqueness problem in the synthesis of critical path  networks representing cognitive tasks. More than one partial order can represent a given comparability graph. The converse of a partial order is obtained by versing  the direction of the ordering relation. Clearly, if a certain partial order is consistent with a comparability graph, then its  converse will be also. The uniqueness of the partial orders which can be constructed depends on the presence of certain sets  of elements called partitive sets. A nonempty set of P of elements is partitive if for every element A (~ P, if X, Y C P then  A and X are comparable if and only if A and Y are comparable. A set of elements is stable if no two of them are comparable.  Shevrin and Filippov (1970) and Trotter, Moore, and Summer (1976) have shown that a comparability graph is consistent with  only two partial orders, one the converse of the other, if and only if every proper nonsingleton partitive subset of the  elements is stable. The number of partial orders consistent with a given comparability graph is discussed in Golumbic (1977,  1980) and Shevrin and Filippov (1970).  The problem of constructing a partial order given comparability information arises in the analysis of reaction times. Multiple arcs between vertices are allowed. To avoid a  possible source of confusion, note that the activities, the elements which are partially ordered, are represented as arcs in  the critical path network, but as vertices in the comparability graph. Suppose activity X precedes activity Y in a critical path network with source s and sink t. If all activities begin execution  as soon as possible, then recall that the slack from X to Y, written s(X, Y), is the amount of time by which X can be  prolonged without delaying the onset of Y. CONCLUSION  In a critical path network in which the activities are represented by arcs, a subnetwork with a unique source, a unique sink  different from the source, and no vertex of attachment other than these can be considered as a superactivity. The arcs of a  superactivity form a partitive set. Furthermore, every partitive set of arcs generates a subnetwork having at least one  superactivity, and if there are several, they are in series. It is easy to see that if P is any partitive set, a partial  ordering compatible with the comparability relation can be found in which the subnetwork generated by P has a unique source, a  unique sink, and no vertices of attachment other than these.  The set of arcs P of a superactivity behave as a single unit in certain respects. If A is an activity not in P, then the slack  from A to all activities of P in the same, and we can speak of the slack from A to the superactivity. Suppose a factor  manipulated in an experiment prolongs more than one activity. If all the activities affected are in a partitive set P which  generates a superactivity, the effects of manipulating the factor will be the same as if it affected a single activity, whose  initial and terminal vertices are the source and sink, respectively, of the subgraph generated by P.  The presence of superactivities affects the uniqueness of the partial ordering of the acrs which can be synthesized, given  which activities are comparable and which are not. Imagine removing a superactivity from the network, reattaching its source  where the sink used to be, and vice versa, and then reversing the direction of all its arcs. The comparability of every pair  of arcs in the new network will be the same as it was in the old network. If the two networks were offered as models for  psychological tasks, then knowing which activities were comparable and which were not would not discriminate between the  networks. On the positive side, though, if no superactivities exist, except for (a) those with only one arc, (b) the critical  path network itself, and (c) those composed of stable sets of arcs, then knowing which activities are comparable and which are  not is sufficient to determine the partial order completely, except for the direction of the ordering relation. Determining  which activities are comparable and which are not is, therefore, extremely informative.  REFERENCES  FISHER, D. L., & GOLDSTEIN, W. M.
