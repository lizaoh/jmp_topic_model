A New Method for Estimating Model Parameters for Multinomial Data 

Richard A. Chechile 

Tufts University 

A new procedure for estimating the parameters of a scientific model is described, and the method is applied and illustrated for the class of experiments with multinominal data structure. The procedure is referred to as the method of population-parameter mapping, and it has a number of novel and advantageous features. The method is a variation of a standard Bayesian analysis. However, instead of directly developing a posterior distribution for the model parameters, this procedure first characterizes the population proportions for the multinomial cells. Random samples are then drawn from the posterior distribution for these proportions, and these samples are mapped to the parameters of the scientific model. This method leads naturally to a definition of model identifiability, and leads to a direct probability estimate of the coherence of the scientific model. Moreover, the new procedure can circumvent the problem of dealing with computationally difficult integrals that frequently occur with Bayesian analyses of complex multinomial models. The method is illustrated by means of several memory measurement models as well as a signal-detection model. � 1998 Academic Press

The statistical problem for any method of estimation is to provide point, interval, and probabilistic information about the model parameters given the available data. Besides achieving these objectives, the current method has at least five additional, desirable properties. First, it performs the estimation within the framework of Bayesian inference (more will be said about this point later). Second, the method can reduce computational time for a class of problems that potentially have exponential time. Third, it provides for insights about model identification. Fourth, it provides for a test of the model itself, even in the case of m=k. Fifth, the method provides for the calculation of the correlations between model parameters. These points will be elaborated upon and illustrated by means of detailed analyses of several psychological models. However, it is helpful first to address some key features and a problem that emerges within the Bayesian framework. 

BAYESIAN APPROACH TO STATISTICAL INFERENCE 

It is not within the scope of the present paper to provide a detailed discussion of the Bayesian�classical debate (for such a discussion see Hartigan, 1983; Lad, 1996; Lindley, 1972; and Stuart 6 Ord, 1991, Chap. 31). However, several general remarks about the Bayesian approach are appropriate here in order to set the stage for the development of the new estimation method. 

Despite the fact that Bayesian inference is uncommon within psychology, it has become widely used in statistics since the 1950s (Stuart 6 Ord, 1991). Although Bayes theorem dates to 1763, it was only in this century that a theory of probability emerged that justified the use of Bayes theorem for statistical inference. The problem with the application of Bayes theorem was that the population parameter was regarded as a constant. If the parameter is considered a constant, then it cannot have a probabilistic description within the context of the classical definition of probability.

However, with the emergence of the concept of subjective probability (De Finetti, 1937�1964), a framework was created for regarding the population parameter as possessing a probability distribution. In this framework, probability is a description of our knowledge concerning the parameter, so even extrinsic-world constants can be treated as possessing a probability distribution in order to characterize our knowledge about the constant. 

Contemporary Bayesian statistics is a philosophically and mathematically rigorous system of inference that obtains point and interval estimates directly from the properties of the posterior probability distribution, cf. Box and Tiao (1973), Hartigan (1983), Lad (1996), Lee (1989), and Press (1989). Nevertheless, there is a technical difficulty that emerges in the Bayesian analysis; namely, it often leads to excessive computational time costs. Dempster (1980, p. 273) states that

The development of the field depends heavily on the preparation of effective computer programs.

POPULATION IDENTIFIABILITY 

New insights regarding model identification can be obtained from the framework of population-parameter mapping. Extant definitions of model identifiability have stressed the relationship between the data space and the space for the scientific parameters, cf. Chechile (1977b), and Dre�ze (1975). Traditionally a model is considered likelihood-identified if there are no two configurations of the model parameters that result in the same set of likelihood functions, see Chechile (1977b). Let (%) i and (%) j be two different configurations of the model parameters expressed as vectors from the space 3 of all possible configurations, and let (xi) be a sample data vector. Following Chechile (1977b) and Dre�ze (1975), (%) i and (%) j are defined as ``observationally equivalent'' iff P((xi) | (%) i)=P((xi) | (%) j) \(xi). Traditionally a model is considered identifiable iff for each (%) j # 3 there is no other (%) j $ that is observationally equivalent to (%) j where j{ j $. Hence, the traditional definition of identifiability is concerned with the question: Is there only one set of model parameter values that produces the same set of likelihood distributions? 

In order to facilitate the discussion of model identification within the framework of population-parameter mapping, the following definitions are useful. 

Definition 1. I mapping. An I mapping (identifiable mapping) is a set of ordered pairs of vectors such that no two ordered pairs have the same first vector. 

Definition 2. UI mapping. A UI mapping (unique identifiable mapping) is a set of ordered pairs of vectors such that no two ordered pairs have the same first vector or second vector. 

For both I and UI mappings, the first vector of each pair is an element in the domain space, and the second vector is an element in the range space. Both I and UI mappings take a point in the domain vector space to a single point in the range vector space (i.e., both types of mappings avoid the one-to-many problem). In this sense both types of mappings are unambiguous. Yet for an I mapping it is possible that two different points in the domain vector space map to the same point in the range vector space, i.e., an I mapping has many-to-one mapping cases. For example, consider the mapping where the domain vector space is the real plane, the range vector space is the nonnegative reals, and the mapping function is the operation of obtaining the distance from the origin. This mapping is an I mapping because for each point in the domain vector space there is a single vector-distance value in the range space. Yet both the points (4, &3) and (3,4) map to the same value of 5 in the range vector space. However, for a UI mapping each point in the domain vector space is mapped to a different point in the range vector space, so the mapping in the example above is an I mapping but not a UI mapping. Other writers have used the term ``one-to-one'' for what is defined as a UI mapping and have simply called an I mapping a ``function.'' Other writers have discussed a particular type of ``one-to-one'' function that also has

POINT ESTIMATION 

Traditionally the topic of point estimation has addressed questions associated with the properties of the estimator (e.g., is it ``unbiased,'' ``efficient,'' and ``consistent''?). While ``unbiased'' and ``efficiency'' are pertinent to classical sampling theory, they are not relevant to a Bayesian analysis. For example, an unbiased estimator, t, of the parameter { must satisfy the condition that E(t)={ where the expectation is across repeated samples. Yet if there were repeated samples, a Bayesian would not ignore the results from those samples (see Lindley, 1972). The prior distribution would capture the results from the previous samples, so it is a meaningless operation to compute the value of E(t) across samples from a Bayesian perspective. In a similar fashion the efficiency of an estimator requires an examination of the variance of the estimator across samples, and this is also a meaningless Bayesian operation. The concept of ``consistency,'' however, is meaningful in both classical sampling theory and Bayesian statistics. If tn is a consistent estimator based on a sample of size n, then for some arbitrarily small positive = and ', there exists some

At any sample size, there will always be some regret or loss in a Bayesian analysis when a full posterior distribution is represented by a single point. The posterior mean, median, and mode are each optimal (depending on the loss function that is to be minimized). The mean minimizes squared-error loss, whereas the median minimizes an absolute-error loss, and the mode minimizes a (0, 1) loss function (see Chechile, 1977b).

A STORAGE-RETRIEVAL MODEL WITH FRACTIONAL AND NO STORAGE 

This model is an example of a model designed not only to separate storage and retrieval components but also to decompose the probability of insufficient storage into fractional versus no storage. Chechile (1990, 1992, 1993) has provided other examples of models of this type. 

The model is designed for a specific task. The task begins with a learning stage. Following the learning stage, there is a free-recall period. The correctly recalled items are not tested further because they clearly are both stored and retrieved. However, the items that are not correctly recalled are probed further, one at a time. The first probe test is a four-alternative forced-choice recognition test (4-afc) with the target embedded among three novel foil items. If the subject is correct in selecting the target, then the testing for that target is done. However, if the subject selects a foil, feedback is provided, and the incorrect foil is removed from the set of forced-choice items. The subject then is tested by a three-alternative forced-choice test (3-afc). If the subject is correct, then testing for that target is finished. If, however, the subject is again incorrect, the incorrect foil is removed, and the subject has a final twoalternative forced-choice test (2-afc). Hence, each target item tracks into only five possible response categories, i.e., (1) correct recall, (2) incorrect recall but correct on the 4-afc test, (3) incorrect recall and incorrect on the 4-afc test but correct on the 3-afc test, (4) incorrect recall and incorrect on the 4-afc and the 3-afc tests but correct on the 2-afc test, and (5) incorrect on all tests for the item. Cells 1 through 5 of a five-cell multinomial correspond to these five response categories.

EXPERIMENT 

In this section, an experiment is described that provides the data needed to illustrate this model. This experiment is part of an ongoing larger study with several other conditions and groups. It is outside of the purpose of the current paper to report additional data beyond the pilot data from one condition. Thus, only the procedural details for this one condition will be reported. 

Method 

Participants. Ten Tufts University undergraduate students participated in the study as part of a research participation requirement. All participants had normal or corrected to normal vision and were fluent in English and were skilled typists. 

Procedure, Materials, and Design. Each participant received two lists of memory items. Each list consisted of 60 stem-target pairs displayed, one at a time, on the computer screen. All the stem items were taken from the computerized version of the Kucera and Francis (1967) word norms. From the entire word corpus, all words between 3 to 7 letters in length and with a word frequency greater than 21 were initially selected. This set of words was further filtered to omit proper names, days, months, numbers, function words, and related forms of other selected words. After filtering, a computer file of 704 words remained as the pool from which all stems and other memory materials were selected. A total of 120 words were randomly selected for use as stem items for the study. The target items were provided by a yoked set of people, who were part of another condition in the experiment which will not be reported here. Because of the yoking procedure, each person had their own set of 120 target items. 

After viewing the stem-target pair, the participants typed the target item. The target item stayed on the screen for 2 seconds after the typing of the word. Upon completion of the 60-item list, the participant had 4.5 minutes to free recall the target items. The computer then selected for each non-recalled target, three random foils from the unused words in the file of 704 words. This random selection was

HYPOTHESIS TESTING 

Hypothesis testing with the method of population-parameter mapping will be considered in three contexts: (1) tests within a single condition, (2) tests of treatment differences between conditions, and (3) tests of the model itself in the multiple-condition experiment. In all of these cases there is an issue associated with the question of applying the model to data on an individual-basis or to the data obtained by pooling across individuals. Multinomial models presume a stable model structure across a set of trials. Depending on the scientific model, it may be either reasonable to pool across individuals or it may be unwise to pool across individuals. Another factor to consider is that some models may result in low P(coh) values for each individual because of a small sample size on that basis, but nonetheless these models may result in a high value for P(coh) on a pooled-data basis. That situation was the case for the storage-retrieval model. 

Suppose that the investigator decides in favor of an analysis on a per-individual basis, and further suppose that there are enough trials to avoid the problem of low P(coh) values due to small sample size. In this case we can simply use the point estimates for the various model parameters as a set of measurements on that individual. Within-condition and between-condition tests of hypotheses can be done on the point estimates by standard statistical methods, e.g., ANOVA, etc. 

However, suppose that the investigator decides in favor of an analysis on the basis of pooling data across the individuals in a common treatment condition. In this case, within-condition tests of various hypotheses are straightforward because the analysis yields cumulative probability distributions for the model parameters. The posterior probability distribution is a direct characterization of the parameter. Hence, probabilistic evaluation of any hypotheses of interest can be directly obtained from the cumulative distribution. For example, from the storage-retrieval analysis, the probability that %F is greater than 0.1 can be directly found to be 0.981. Bayesians do not evaluate any hypotheses concerning a parameter being precisely equal to a specific point, because all hypotheses of this type have a probability of 0.0. However, one can examine the probability that the parameter resides in a small interval. For example, there is a probability of 0.111 for the hypothesis that %S is between 0.680 and 0.685. Moreover, if the investigator is interested in detecting differences for a specific parameter among the I treatment conditions, then the I

Let us now consider the assessment of the model itself across treatment conditions. It is easily proved that with I treatment conditions, the overall value for P(coh) is the mean of the I separate values for P(coh). Because the prior analysis for P(coh) is identical for each condition, the overall value for p(coh) is the mean P(coh) value divided by the no-data value for P(coh). Both the overall P(coh) and \(coh) values are a direct means to assess competing psychological models. Furthermore, the investigator may be interested in any large changes in P(coh) between the conditions. Perhaps the model does not work as well in some treatment conditions. By understanding the nature of the experimental treatment and by examining the decomposition of model failures, a clearer picture can be constructed about the limits of the model. 

DISCUSSION 

Although the original purpose for the method of population-parameter mapping was to circumvent a computational complexity problem that frequently occurs with a Bayesian analysis of complex multinomial models, other important benefits were also achieved. The new definition of model identifiability, the direct probability assessment of model coherence, and the calculation of the correlational structure among the model parameters are important features of the new estimation method. These positive features are justification enough for population-parameter mapping even if there were no saving in computational ease. Also the method shares with a standard Bayesian approach the benefit of generating an entire distributional description of the model parameters. 

However, some researchers might be concerned about the requirement of using only the coherent random samples. In order to address this concern, let us first consider several reasons for low coherence. 

Suppose that the set of points in ,-space, which are consistent with the model constraints, is the empty set because the coherence requirements are contradictory.

For example, consider the model where ,1=%, ,2=2.5&2%, and ,3=%&1.5. This model is hopelessly flawed because of mutually contradictory coherence requirements, i.e., from cell 1 we require that 0�%�1 but from cell 3 we require that 2.5�%�1.5. It is a mistake to estimate the parameter in this model. 

Suppose that a model is not contradictory, but a few of the samples from the posterior distribution are coherent. This result could occur for a number of reasons. A small-sample experiment could result in a broad posterior distribution that is not very different from that of the noninformative prior. In such a case the coherence probability would not be expected to be much larger than the probability of coherence found in the analysis of the prior. The remedy for having too small a sample size is to obtain more data. However, if there still is a low value for P(coh) when the sample size is reasonably large, then it would appear that the model is not highly creditable. Again, it is not sound science to estimate the parameters for an unreasonable model. The estimates are measurements, and measurements based on an incorrect model are misleading. Rather than estimating the model parameters in such cases, it would be more efficacious to revise the model. Moreover, the task of revising the model can be made easier by an examination of the reasons why the model is failing. Perhaps in the case of an unsuccessful model, the model is having difficulties with one of the absurdity conditions.[10] Hence, the restriction of only using the coherent samples from the posterior distribution is a highly desirable feature of the estimation method. 

The feature of using only the coherent samples is also relevant in addressing the question as to whether P(coh) can meaningfully be computed in a standard Bayesian analysis. If a standard Bayesian analysis computed the probability that the multinomial vector (,1 , ..., ,k) was in the region not permitted by the scientific model, then the probability of P(coh) could also be computed without using population-parameter mapping. However, that analysis would not be consistent with the Bayesian analysis that sets the prior probability for that absurdity region equal to zero. Hence the same Bayesian analysis cannot simultaneously compute a P(coh) value and estimate the parameters contingent on the full acceptance that the scientific model is correct. Should such inconsistent results be reported nevertheless, then the two Bayesian analyses would themselves be incoherent. However, with population-parameter mapping there is no inconsistency because only the subset of sampled points in the permissible ,-space region is used for parameter estimation. 

It is also natural to speculate about a possible classical counterpart to the method of population-parameter mapping. There are difficult obstacles in the way of achieving such a classical counterpart. With population-parameter mapping, the point in ,-space is a k-dimensional random variable, so it is appropriate to take random samples from the posterior joint distribution. Furthermore, a strict classical position should follow the position, perhaps best expressed by von Mises (1957, p. 11), i.e., ``in order to apply the theory of probability we must have a practically unlimited sequence of uniform observations.'' Because the ,-space representation, within the classical framework, is a k-dimensional vector of constants, there cannot be an unlimited sequence of uniform observations for the coherence of the model. The von Mises' position would, for example, result in either the model being or not being coherent. From this classical framework, there is no relative frequency associated with the coherence of the model, except across repeated experiments (i.e., different data sets). Given these fundamental differences between the classical and Bayesian inference systems as to what are the variables and what are the constants, it is not clear how a principled classical analysis might be developed to deal with the type of inference described in this paper. 

Finally, although this paper has limited the scope of population-parameter mapping to the class of scientific models for multinomial data, it is possible that the estimation method could be generalized to other data types (e.g., process models of Gaussian data). However, the same type of theoretical development that was addressed in this paper would need to be resolved before such a generalization could be established. For example, the counterpart to Theorem 1 would need to be established as well as the method for sampling from the exact posterior distribution. At present there is no clear reason why the extension of population-parameter mapping could not be established for the estimation of scientific model parameters of other data types.
