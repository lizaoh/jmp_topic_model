With  Z the number of choice probabilities, (3) defines a convex cone in R1. The relative frequencies Qa,B = m(a, B)IN(B) give a  point Q in R1 lying outside the cone if the Q,,,B do not satisfy (3). Thus the quadratic program has to find that point inside  the convex cone which satisfies (4) and minimizes X2. The algorithm used in the program has been designed by Beale (1967) and  is based on the simplex procedure from linear programming. There are two major steps in this algorithm: The first is finding  any solution to (3) and (4), the second is the minimization of (2). The first step is performed by the usual simplex method of  solving any system of linear inequalities. It results in a "basic" solution-an extreme point of the convex polyhedron defined  by (3) and (4), where at least l constraints hold as equalities. During the second step the partial derivative of X2 at each  basic point is computed and if it is negative for some variable then this variable is increased until one of the presently  noneffective constraints becomes effective or until the partial derivative for this variable becomes zero. In the first case  the usual simplex procedure starts to increase a new variable. In the second case an auxiliary constraint is introduced  providing a path along which the simplex algorithm can move by subsequently increasing those variables whose partial  derivative indicate a minimization of X2. The program is terminated if the optimum of the partial derivative of X2 for each  variable within the original constraints has been found (see Land & Powell, 1973, pp. 21). Since (2) is a convex function we  can be sure to find a unique minimum.  The data used in this paper have been generated by simulation according to Eq. (1). For further use of programming methods on estimation problems it must be clear that the constraints have to be a solvable  system of linear equalities or inequalities. The reason is that the set of solutions to the constraints has to be convex. In a  psychological measurement context we often have constraints that are linear or that can be made linear by some transformation  (for example, the multiplication condition of the B.T.L.-model by using the "logit"-transformation, Suppes & Zinnes (1963)).  For these models quadratic programming provides a valuable tool for parameter estimation, because it can handle simple bounds  and order restrictions on the parameters as well as complex systems of linear inequalities.
