JOURNAL OF MATHEMATICAL PSYCHOLOGY 28, 401-420 (1984)  Estimating Response Time Hazard Functions: An Exposition and Extension  BRUCE BLOXOM  Vanderbilt University  The hazard function of a response time distribution describes the temporal dimension of a response process and completely  characterizes the distribution. The function can be estimated by a quadratic spline which is constrained to have one of a  number of simple functional forms. The estimator can also be employed when the response time is an unobserved component in a  parallel or serial pair of processes. 1984 Academic Press, Inc.  The problem addressed by this paper can be stated as follows: Assume that we have a process or a set of processes, initiated  by the presentation of a stimulus (e.g., a pair of figures) and terminated by the subject's error-free choice of a response  (e.g., indicating whether the figures are the same or different); the time elapsed between the stimulus presentation and the  response selection is a random variable which is referred to here as the response time and is assumed to be independently and  identically distributed over replications of each stimulus-response sequence; the problem is to describe the temporal  dimension of the response process while completely characterizing the distribution of the response times. Although other  dimensions of the response process are important (e.g., errors), they are not within the scope of this paper.  There are two ways of approaching this problem. One approach is to develop a stochastic model of the response process based on  plausible assumptions about the process. If the model is fully developed, it specifies a parametric family for the  distribution of response times and, thus, completely characterizes that distribution. Furthermore, if the data are consistent  with the model's predictions, estimates of a few parameters can be used to describe the response process. Interesting examples  of the recent application of this approach are found in Luce and Green (1972) and Ratcliff (1978, 1981).  An alternative approach to the problem as defined here is to describe some important features of the response process without  initially invoking a strong set of assumptions. This approach can be useful when detailed aspects of the process are not  sufficiently well understood to justify strong assumptions, e.g., when the process involves the formation of a gestalt. However, two problems have been noted  regarding work with such functions. First, estimates of some regions of the functions have very large sampling errors. Second,  it is not always clear how to interpret what some regions of the functions tell us about the response process, e.g., it is not  clear how one interprets the skew or kurtosis of a response time distribution (Ratcliff, 1978).  The purpose of this paper is twofold. The first purpose is to outline an approach which has been previously suggested for both  describing the temporal dimension of a response process and completely characterizing the distribution of response times  without making strong assumptions about the nature of that process. Because using this approach requires estimating a  distribution, the second purpose of the paper is to describe a method which can be used for such estimation. The method is  useful in three contexts: (1) when the response times of the process of interest are directly measurable, (2) when the  measured response times include component processes (e.g., anticipation) which are operating in parallel with the process of  interest, and (3) when the measured response times include component processes (e.g., motor reaction time) which are operating  in series with the process of interest (e.g., decision time). Bloxom (1984) provides a more complete development of the use of  the method in the first of these contexts; the method has not been previously proposed for use in the second context; the use  of the method in the third context is also new and is an alternative to previoulsy proposed approaches (Bloxom, 1979, 1983;  Mendelsohn & Rice, 1982; Rice & Rosenblatt, 1983). This paper illustrates the results of applying the method in the first  context. The implementation of the method in the second and third contexts is not illustrated but is straightforward. Thus,  the reader can employ and study the method on any appropriate data which are available.  HAZARD FUNCTIONS  The concept used here to describe the temporal dimension of a response process is the hazard function which is associated with  the distribution of response times. This is a continuous function which describes the rate at which responses are produced in  each instant following stimulus presentation. The function is important in that it provides descriptive information which is  not provided by either the density function or the cumulative distribution function alone. Also, it is a function which  completely characterizes the response time distribution, making the function useful for evaluating the appropriateness of  plausible alternative parametric families of distributions. Each of these points will now be developed further.  Definition of a hazard function. A hazard function can be defined in the following stepwise way. Begin by considering a very  constrained response process such that, in every instant following stimulus presentation, there is a constant likelihood of  the response occurring, given that it has not yet occurred. This is an assumption of a stationary Poisson process and leads to  an exponential response time distribution  RESPONSE TIME HAZARD FUNCTIONS 403  which has a rate parameter equal to the (constant) conditional likelihood. Examples of the use of this process can be found in  Luce (1959), McGill and Gibbon (1965), Luce and Green (1972), and Ashby and Townsend (1980).  Next, following Luce (1959) and McGill (1963), consider a response process which is much less constrained than the Poisson  process. Assume that, following stimulus presentation, the conditional likelihood of a response can change as a function of  time. For example, the conditional likelihood of a response may be a monotone increasing function of time, i.e., the longer  the subject goes without responding, the greater may be the likelihood of his responding in the next instant (Ratcliff, 1978).  Alternatively, the conditional likelihood may increase and then decrease to an asymptote if some aspect of the stimulus  presentation is terminated or fades prior to the subject's response (Burbeck & Luce, 1982). Other examples of response time  conditional likelihood functions are developed in Anger (1956), Kelleher, Fry and Cook (1959), Thomas (1971), Townsend and  Ashby (1978), and Ashby (1982). Besides its use in psychology, this function has been employed to describe failure rates of  mechanical systems and mortality rates in human populations (Barlow & Proschan, 1975).  In statistics, the conditional likelihood function just described is termed the hazard function of a distribution, following  its use to describe failure rates of mechanical systems. Hazard functions of parametric families of distributions. Because a hazard function completely characterizes a distribution of  response times, an estimate of that functon can be useful not only for describing the temporal dimension of the response  process but also for ruling out consideration of plausible alternative models or parametric families of distributions. The  estimated hazard function may have a functional form which is not consistent with the hazard functions of those models or  distributions. For example, Burbeck and Luce (1982) found that the estimated hazard function for auditory signal-offset  detection increased and then decreased to a level asymptote, which is not characteristic of the hazard functions of a number  of commonly considered response time distributions, e.g., the gamma and Weibull distributions. Also, the upper tail of the  estimated hazard function did not behave in a way characteristic of the often-posited Wald (inverse Gaussian) distribution.  Thus, models which produce these distributions could be viewed as not providing an accurate description of the process  generating the response times.  Clearly, such a use of a hazard function requires knowledge of the functional form of the hazard function associated with each  model or distribution under consideration. Fortunately, there is a substantial body of research on hazard functions associated  with various parametric families of distributions. The hazard functions of most of these distributions can be placed in one of  four classes: increasing (I), decreasing (D), increasing-then-decreasing (ID), and decreasing-thenincreasing (DI) functions of  time; in this classification, both increasing and decreasing functions can also include constant functions of time. In practice, however, the appropriate set of constraints is unknown because the true form of the  hazard function is unknown. This presents two problems: choosing plausible alternative sets of constraints to consider and  choosing an objective criterion to be optimized and employed for selecting among alternative sets of constraints.  In the absence of prior knowledge about the process which is generating the response times, sets of constraints given in  Bloxom (1984) constitute a reasonably diverse list of plausible alternative sets. Their use can result in hazard estimators  which are increasing, decreasing, increasing-then-decreasing, or decreasing-thenincreasing and which have no, one, or two  inflections. However, it is possible to conceive of processes which generate hazard functions not included in these classes of  functions. For example, if fast guessing times are mixed with stimulus processing times, the response time hazard function can  be increasing-then-decreasing-thenincreasing (again). When such processes seem plausible, sets of constraints which are  descriptive of the corresponding hazard functions need to be added to the list of alternative sets of constraints.  Selecting among the plausible alternative sets of constraints requires the choice of a criterion and a test statistic which  are optimal under that set of constraints which is descriptive of the true distribution's hazard function. The likelihood  ratio criterion is intuitively appealing here, particularly when the coefficients of the spline are estimated by maximizing  the likelihood or log likelihood function as described in the previous section. However, it should be noted that the  chi-square distribution which is usually employed in statistical tests of the likelihood ratio cannot justifiably be employed  here. The problem is that, because of the inequality constraints being placed on the parameter estimates, the effective number  of parameters used in computing the chi-square is a random variable. There is sampling variability in which ones of and how  many parameter estimates lie on the boundaries defined by the constraints; whenever a boundary is encountered by a constraint,  the same parameter estimates are obtained as would be obtained if the constraint were set equal to its boundary value a priori  and, therefore, as would be obtained with a smaller effective number of parameters. This does not necessarily mean that a test  statistic cannot be developed; it only means that defining a critical value for a test statistic requires further study. NECESSARY METHODOLOGICAL RESEARCH  The main body of this paper has suggested alternative approaches (weighted least squares and maximum likelihood) for obtaining  a quadratic spline estimate of a response-time distribution's hazard function in a variety of contexts: for a single  distribution and for a distribution embedded in a parallel or serial model with assumed independence of components. Although  constrained splines can provide good approximations to known hazard functions (e.g., see Fig. 6) and can be estimated from  sample data by using the alternative approaches discussed here, it is not clear how precise the sample-based estimates are.  Also, as noted earlier, it is not clear how one can make statistical tests of hypotheses which are stated in terms of splines  with inequality constraints.  The question of the precision of the spline estimates has two important and related aspects. One aspect pertains to the  amounts of bias and sampling error associated with estimates obtained using realistic sample sizes. Research with other  monotonic estimates of hazard functions suggest that such estimates have much less sampling error than do their unconstrained  counterparts (e.g., Bartoszynski et al., 1981). We now need to know if reasonably accurate hazard function estimates are  likely with sample sizes as small as those which are often used in psychological studies.  RESPONSE TIME HAZARD FUNCTIONS 417  Bloxom (1984) found a useful degree of precision in the spline hazard estimator of a single distribution with a sample size as  small as 150. Further study is needed to see if this also obtains when the distribution is embedded in a model of parallel or  serial processes.  A second aspect of the question of the spline's precision pertains to how one might estimate that precision in practice, i.e.,  how to place confidence intervals around points on the spline estimate. In dealing with this problem, maximum likelihood  estimates are often useful because they are asymptotically normally distributed with easily estimable standard errors.  However, as was noted earlier, the usual asymptotic features of maximum likelihood estimation cannot be assumed to hold when  inequality constraints are placed on the parameter estimates. The problem here is that the constraints truncate the  distributions of the estimates, making them nonnormal and invalidating the estimates of the standard errors obtained from the  maximum likelihood procedure. Resampling methods such as the bootstrap (Efron & Gong, 1983) provide possible alternatives to  the use of maximum likelihood asymptotic theory for obtaining confidence intervals. However, their practicality depends in  part on rapid computation of parameter estimates. Again, it is important to note than computation can be more rapid with  weighted least squares procedures than with maximum likelihood procedures.  SUMMARY  The purpose of this paper has been to describe how estimating the hazard function of a distribution of response times is  useful for describing the temporal dimension of the response process, for characterizing the distribution of response times,  and for thereby eliminating further consideration of inappropriate parametric models of response process. A  previously-developed hazard estimator was suggested for use when plausible alternative constraints are being considered for  resricting the form of the estimator. Also, the estimator was further developed for use when the distribution of interest is  an unobserved component of a parallel or serial pair of processes.  Although questions were raised regarding the precision of estimates provided by the method, there is no reason to suppose that  these questions are unanswerable or pose major difficulties for the utility of the method. Further study is needed to point  out contexts in which the method provides precise and informative results.
