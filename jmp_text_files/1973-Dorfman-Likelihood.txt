There are a wide variety of problems of contemporary  psychophysics for which the assumption of a continuum of sensory states seems appropriate, and this family of learning models  has a natural extension to such problems. One potential difficulty in the application of such models to data is that  estimation of parameters requires the use of numerical procedures when the method of maximum likelihood is used. Given a  likelihood function generated from an additive model, this paper gives sufficient conditions for log-concavity and strict  log-concavity of the likelihood function. If a likelihood function is strictly log-concave, then any local maximum is a unique  global maximum, and any solution to the likelihood equations is the unique global maximum point. Consequently, it would be quite valuable to know if one  has converged upon the global maximum, or how many other relative maxima exist. When a likelihood function has more than one  maximum, difficulties can arise since there is usually no way of selecting the solution nearest to the vector of population  parameters (see Kraft and LeCam, 1956; Billingsley, 1961; Kendall and Stuart, 1961, p. 61). Also, one often attempts to find  the maximumlikelihood estimate by solving the system of likelihood equations-that is, by finding a point at which the first  partial derivatives of the logarithm of the likelihood function vanish. There are cases in which the only solution to the  system of likelihood equations is a saddle point (see Solari, 1969). Given a likelihood function generated from an additive  model, this paper gives sufficient conditions for the likelihood function to be log-concave and strictly log-concave. A  function is defined as log-concave (strictly log-concave) if the logarithm of the function is concave (strictly concave). If a  likelihood function is strictly log-concave, then any local maximum is a unique global maximum (see Kowalik and Osborne, 1968,  pp. 138-141). Further, given that the likelihood function is strictly log-concave, any solution to the likelihood equations is  the unique global maximum point. These conditions are quite easy to evaluate in particular cases, and hence, the results to be  presented are quite useful.
