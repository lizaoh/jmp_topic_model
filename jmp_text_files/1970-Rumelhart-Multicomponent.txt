This model  is applied to data from whole report, partial report, detection, and backward masking experiments. On the whole, the results  are encouraging. In a final section the model which is presented as an extension of Bower's multicomponent model for memory is  shown to be derivable as a limiting case of LaBerge's neutral element stimulus sampling model.  In a recent paper, Bower (1967) has proposed a multicomponent theory of the memory trace. In his theory Bower assumes that  each to-be-remembered item is initially encoded as a list or vector of the attributes or features of the physical stimulus.  During forgetting, he suggests that the values of some of these components are erased or become blurred. Although Bower's  theory appears to account well for a wide variety of memory phenomena, he does not discuss the way in which his memory vector  is derived from the physical stimulus.  The present paper discusses the time course of the construction of the memory vector. Since about 1960, numerous studies using  tachistoscopic presentations have investigated the temporal properties of perception (e.g., Averbach and Coriell, 1961;  Eriksen and Collins, 1965; Estes and Taylor, 1964; Sperling, 1960; for a general review see Kahneman, 1968). This paper gives  a systematic account of these data in the context of Bower's multicomponent conception of the memory trace. In a related  paper, Norman and Rumelhart (in press) show how the perceptual system outlined here can be extended to cover auditory as well  as visual presentations and the way in which the perceptual and memory models interact.  The overall theoretical view is illustrated in Fig. 1. The information extraction itself is assumed to be based on an analysis of the physical features of the elements of the  display. Each of the N positions of the stimulus display is assumed to act as an independent input channel. All features  extracted from a particular channel are fed into counters-one associated with each of the possible stimulus inputs. Whenever a  feature is extracted in a given channel, all of the counters in that channel associated with possible inputs consistent with  that feature are incremented. Furthermore, it is assumed that any c features uniquely specify any of the possible inputs.  (Thus, the number c would be expected to depend on the set of possible inputs.) This assumption allows us simply to count the  number of features extracted in a given channel; as soon as the count equals or exceeds c we know that the stimulus can be  uniquely recognized. This view is roughly analogous to Selfridge's (1959) "Pandemonium" model for pattern recognition. The  information in the VIS corresponds to Pandemonium's "image demons"; the feature extracting mechanisms correspond to the  "computational demons,'"' the counters to the "cognitive demons," and the criterion, c, to the decision rule employed by  Pandemonium's "decision demon." Redundant Critical Elements  In their second paper, Estes and Taylor (1966) introduced an important variation of the detection procedure. Redundant  critical elements were added to some of the displays. Some displays contained only a single critical element as before, but  some contained two critical elements and some four. The model predicts a very simple relationship between the probability of a  correct detection and the number of critical elements in the display. The theory predicts that each element of the display is  processed independently. (A proof of this assertion is given in the Appendix, Theorem 2.) Response Times  Another measure of theoretical importance is response time. Estes and Wessel (1966) and Wolford et al. (1968) analyzed  response times as a function of display size and of the number of redundant critical elements in the display. Their major  results can be summarized this way:  A. Times for incorrect responses. (1) are longer than for correct responses; (2) do not change as display size and number of  critical elements in the display are varied.  B. Times for correct responses. (1) get longer with increases in display size; (2) get shorter with increases in the number of  critical elements.  C. Times corrected for effects of guessing. (1) get longer with increases in display size; (2) do not change with increases in  the number of critical elements.  Let us now see how the present model handles these results. We assume that when the subject detects a critical element he  makes the appropriate response. If he fails to detect a critical element by the time that the information in VIS has faded  beyond some criterion point, he guesses. Under these assumptions it is clear that mean response times for incorrect responses  should always be longer than those for correct responses. Moreover, the incorrect response times should not vary with  variations in experimental parameters like display size and number of redundant critical elements.  Results B1 and B2 follow directly from the fact that guessing responses take more  204 RUMELHART  time than true detections. As display size decreases, or the number of redundant critical elements increases, the proportion  of correct responses which are accounted for by guesses decreases; hence, response times decrease.  The argument for C l is more complicated. This result follows from the fact that as the intensity parameter for a Poisson  process increases (as it is assumed to do with a decrease in display size), the waiting time to the kth event decreases.  However, the same logic applies for C2 when there is an increase in the number of redundant critical elements. It can be shown  that as the number of critical elements increases, the mean response time for detection of a critical element should  monotonically decrease. Thus, this theory agrees with all of the results except C2. It is not entirely clear, however, how  much of a decrease in response time is predicted. It is entirely possible that the predicted decrease is too small to have  shown up in an experiment. Before concluding this section on detection experiments, one more result should be mentioned. Yellot and Curnow (1967) report  that in a detection experiment, with a set of three critical elements, subjects were unable to do better than chance on second  guesses following an incorrect response. That is, subjects lacked any partial knowledge of elements that they failed to  detect. Yellot and Curnow interpreted this result as favoring an "all-or-none" detection model. The theoretical results  reported here (i.e., the fact that cd was estimated to be about 1.0) support this conclusion. However, it seems probable that  as the difficulty in discriminability or number of elements in the "critical element set" increases, the subject would need  more features on which to base his response, and under these conditions second guesses might well contradict the "all-or-none"  assumption.  BACKWARD MASKING  Backward masking experiments form still another important line of evidence on the temporal characteristics of the pattern  recognition process. The typical experimental procedure of these experiments involves the presentation of a target stimulus  followed, after some delay, by a masking stimulus. Two types of results have been reported in the literature. Under some  conditions the probability of detection varies  206 RUMELHART  monotonically with the delay of the masking stimulus-the greater the delay the greater the probability of detection (cf.  Alpern, 1953; Eriksen and Collins, 1964, 1965; Kolers, 1962). In other experiments the greatest degree of masking has been  found to occur with delays of from 20 to 100 msec after the offset of the target stimulus (cf. Alpern, 1953; Averbach and  Coriell, 1961; Kolers, 1962; Kolers and Rosner, 1960; Mayzner et al., 1965; Weisstein, 1966; Weisstein and Haber, 1965;  Werner, 1935). Kolers (1962) discussed some of the conditions which lead to nonmonotonic masking functions.  No one has provided a complete theoretical account of these data. Some of the simple masking functions, however, do make  considerable sense in the context of the multicomponent model. Perhaps the simplest masking phenomena occur when the target  stimulus consists of a single letter and the masking stimulus consists of a homogeneous flash. Thompson (1966) has studied  this situation extensively. Another seemingly paradoxical experimental result which can be explained in terms of the present theoretical formulation was  reported by Fehrer and Raab (1962) and replicated several times (Fehrer and Biederman, 1962; Harrison and Fox, 1966; Schiller  and Smith, 1966). These investigators found that reaction times to phenomenally nonexistent stimuli in a backward masking  experiment were indistinguishable from those to phenomenally present stimuli. If we supposed that a single component of the  target is sufficient to initiate a response, Fehrer and Raab's result would be predicted. We would expect, however, that if  the correct response were made contingent on details of the target stimulus (e.g., in a choice reaction time paradigm) more  than one element would be necessary to initiate the correct response and reaction time should be longer to masked than to  unmasked targets.  Before concluding this section, it should be mentioned that the interpretation given here to backward masking effects is in no  way inconsistent with the suggested relationship between backward masking and apparent motion (beta motion) (cf., Fehrer,  1966; Fehrer and Raab, 1962; Fehrer and Smith, 1962; Kahneman, 1968; Kolers, 1963). It is completely plausible that beta  motion is perceived when a partially processed figure is replaced by a spatially displaced similar figure. If this were true  the observed similarities between the conditions under which backward masking and apparent motion are observed (Fehrer and  Raab, 1962; Fehrer and Smith, 1962; Kahneman, 1968; Kolers, 1963; Kolers and Rosner, 1960) would be expected.  RELATED THEORETICAL FORMULATIONS  Although the model discussed in this paper was presented in the context of Bower's (1967) multicomponent theory of memory,  there are other general theoretical formulations which are equally compatible with these ideas. The relationship to  PERCEPTION OF VISUAL DISPLAYS  stimulus sampling theory will be spelled out; other related theories will simply be mentioned in passing.  The relationship of the multicomponent model to stimulus sampling theory is very direct. In fact, under the appropriate  translation of terminology, the present model can be considered a natural extension of LaBerge's (1959, 1962) recruitment  theory. LaBerge, in the tradition of stimulus sampling theory (cf. Estes, 1959a, b), assumes that a stimulus presentation  consists of making available to the subject a set, S, of stimulus elements. For each of the possible response alternatives  there is a subset of S, denoted S; for the jth response alternative, which is associated with (i.e., conditioned to) that  response. The remaining stimulus elements which are associated with none of the response alternatives form a subset of neutral  elements denoted So . On a presentation, a subject is assumed to sample the stimulus elements of S sequentially with  replacement. At the time he finds the jth element associated with any one of the responses he is assumed to make that  response. LaBerge assumes that time A is consumed by the sampling of each element. CONCLUSION  The object of this paper has been to attempt, within the context of Bower's (1967) multicomponent theory of the memory trace,  to bring together in a systematic way  PERCEPTION OF VISUAL DISPLAYS  213  the current literature on tachistoscopic perception. The multicomponent theory of the pattern recognition process outlined  here has proven useful in the analysis of three different types of experimental procedures: Sperling's partial report  experiments; Estes' detection experiments and backward masking experiments. Although further refinements may be necessary,  especially in the complex masking experiments, the results have been very encouraging.  The model is especially interesting with regard to the often-asked question of whether visual processing is serial or parallel  in nature (cf., Averbach, 1963; Estes and Taylor, 1964, 1966; Sperling, 1963, 1967; Weisstein, 1966; Wolford et al. 1968). The  present model implies that neither view is entirely correct; the processing is parallel in that all elements of the display  are worked on simultaneously. However, features are extracted one at a time and items are recognized and transferred serially  into the STM store.  The next step in the development of the model is to integrate this perceptual system into the memory system to account for  data which certainly involves both perceptual and memory components (Aaronson, 1966, and Yntema, Wozencraft and Klein, 1964).  This next step is carried out in another paper (Norman and Rumelhart, in press).  APPENDIX  THEOREM 1.
