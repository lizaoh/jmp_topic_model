Anderson’s brain-stale-in-a-box (BSB} neural network model may be viewed as a version of Hopfietd’s two-state neural model with continuous rather than discrete stales and synchronous updating. A generalization of the BSB model which allows each model neuron in the system to have a bias and its own maximum and minimum firing rate is considered. Morcover, the generalization does not require that the matrix of connection weights be symmetric. A method of choosing the step constant of the algorithm is then described which guarantees that the generalized discrete-time BSB model: (i) approaches the largest set of system equilibrium points, and {ii} minimizes the Bolizmann machine quadratic energy function. A simple test (essentially a vector by matrix multiplication) for checking for asymptotically stable equilibrium points is also provided. Finally, a special case of the generalized BSB model is shown to be applicable to a particular type of optimal control problem. = 1993 Academic Press, Inc. 

Anderson, Silverstein, Ritz, and Jones (1977) proposed an interesting neural network model of categorization known as the brain-state-in-a-box (BSB) model. This simple model consisted of a set of d model neurons or units which were symmetrically interconnected and fed back upon themselves. Each unit at each instant in time simultancously computed a weighted sum of its inputs and then used this weighted sum to update its activation value. In addition, a simple nonlinearity was introduced so that the activation value of cach unit remained bounded between its maximum and minimum activation level. The state of the neural network was represented as a pattern of activation over the neuronal units which was amplified if the activation pattern was “familiar” to the system, and squelched if the activation pattern was not “familiar.” Golden (1986a) provided an analysis of the behavior of the BSB model in discrete-time and discussed explicit conditions which would guarantee that the BSB model decreased the value of a specific “energy” function for all nonstable

In this paper, Anderson’s BSB model (Anderson, Silverstein, Ritz, & Jones, 1977) will be generalized in a variety of different ways. A global convergence theorem for this generalized algorithm in discrete-time is presented following the approach of Golden (1986a). A simple test (essentially a vector by matrix multiplication) for checking for asymptotically stable equilibrium points is also presented. The results in this paper provide a better understanding of applications of the BSB model by considering a related algorithm where the matrix is not symmetric, the case where the BSB modei has “biases,” and the case where the maximum and minimum “firing rates” of the units in the model can be different for each unit in the system. The resulting generalized BSB model is called the generalized-brain-state-in-a-box (GBSB) model and was developed in order to deal with an optimal control problem associated with modelling story comprehension and recall considered by Golden and Rumelhart (1991, in press). Before describing the general issues which confronted Golden and Rumelhart (1991, in press), it is worth noting that the optimal control problem discussed here is fairly general and should be applicable to many psychological domains such as speech perception and motor control (e.g., Jordan, in press, Barto, 1989). 

Golden and Rumelhart (1991, in press) approached the problem of modelling story comprehension and recall within the framework of a high-dimensional state Space known as “situation” state space. A point in situation state space consists of a collection of m facts about the world, each of which can be classified as either being “present” or “absent.” The reader’s current mental state is therefore modelled as a single point in situation state space. At some later point in time, the reader’s mental state is modelled as another point in situation state space and it is convenient to order these points according to a time index. Thus, the “unfolding” or “evolution” of the reader’s mental state as a function of time, may be represented as an ordered sequence of points or trajectory in situation state space. Some trajectories in situation state space will be more likely to occur than others. Accordingly, the reader’s world knowledge is modelled as a probability mass function which assigns a nonzero probability to every trajectory in the situation state space. Within this framework, the problem of story comprehension is formally viewed as finding a trajectory in situation state space which is most probable with respect to the reader’s world knowledge (ie., the reader’s probability mass function).

OVERVIEW OF THE THEORETICAL ANALYSIS AND RESULTS 

As previously noted, the GBSB model is a complex nonlinear dynamical system and so conventional methods of linear analysis are not applicable. For example, one could try to “linearize” the GBSB model using Liapunov’s first or indirect method (Luenberger, 1979, pp. 324-328) in order to convert the GBSB model into 

288 

RICHARD M. GOLDEN 

a linear system over certain regions of the state space, but this approach will not work since the nonlinearity is continuous but not differentiable. Another approach is to attempt to apply a generalization of Liapunov’s second or direct method known as the Invariant Set Theorem (Luenberger, 1979, pp. 332-349). An invariant set is simply a region of the state space with the property that if the state of a specific nonlinear dynamical system enters this region, then the system state can never leave the invariant set. Perhaps the simplest example of an invariant set is an equilibrium point which is defined to be an invariant set consisting of exactly one point. This method of analysis for neural networks was originally popularized by Hopfield (1982) and is essentially based upon the following ideas. 

First, the goal of completely analyzing the nonlinear dynamical system is abandoned in favor of making some weaker statements about the system’s behavior. In particular, the goal of an Invariant Set Theorem type of analysis is to characterize the dynamics of the system in terms of the system’s equilibrium points (more technicaily, the system’s “invariant sets”). This is done by proving that all sequences of activation patterns generated by the model converge to the set of system equilibrium points and then developing criteria for deciding if a given equilibrium point is an “attractor.” An attractor equilibrium point or equivalently an asymptotically stable equilibrium point has the property that if the system state is initiated sufficiently close to such an equilibrium point, then the system state will always converge to that equilibrium point. For example, consider modelling the dynamics of a turbulent river. Although the detailed dynamics of the water flow will probably never be weil understood, important insights into its behavior can be obtained by identifying the location of the vortices of the river. These vortices correspond to the asymptotically stable equilibrium points of an Invariant Set Theorem type of analysis. 

Proving the existence of invariant sets and showing exactly which invariant sets will be attractors remains a difficult problem. For this reason, the Invariant Set Theorem is based upon the idea of a “summarizing function” (Luenberger, 1979, pp. 319-320) which summarizes the nonlinear dynamical system’s performance. Some heuristic arguments which provide intuitions about how a version of the Invariant Set Theorem is applicable in the analysis of the GBSB model are now provided.
