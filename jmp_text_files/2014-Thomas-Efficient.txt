First, we present an efficient algorithm to compute AMI using kernel density estimation that incorporates the current best practices for using this type of estimation technique on univariate and multivariate data ( Scott & Sain, 2004 ). We emphasize here and later that one of the primary attributes of value for our offering is that the function embodying this algorithm be (very) usable by the jobbing experimentalist. All too frequently, computational advances within mathematical psychology are not widely embraced by the nontechnical community, a fact that, in our view, limits the impact these advances can have. Some in our discipline have taken on the challenge of bringing these advances to a greater audience through emphasis on approachability rather than sheer technical acumen (e.g.,  Wagenmakers, van der Maas, & Grasman, 2007 ). Next, we compare our algorithm to the quality of several other AMI estimation functions based on Eq.  (2)  or  (3)  that are in circulation, are relatively easy to use, and are efficient in terms of computational speed. These tests include, (i) examining the bias and consistency of each algorithm in a known case, e.g., bivariate Gaussian random variables, (ii) investigating each algorithm’s ability to “see through” marginal density information that is irrelevant for AMI by exploiting the relationship between copula entropy and mutual information, and (iii) determining the relative ability of AMI algorithms to capture the best time-delay for embedding a time-series in reconstructed phase space. Throughout, the impact of choices that are necessary in many forms of density estimation (i.e., bandwidth), and the effects of sample size on the quality of the estimators are explored. Overall, results indicated that our new AMI algorithm provides a highly accurate and usable option, as compared to the others, for smaller length time series pairs. We close with an illustration in which we compute AMI between measured EEG signals to show one type of application of the construct that may interest readers.

In addition to recasting the kernel estimate in a simpler form, we employ a common matrix technique to compute the summations in Eqs.  (3) ,  (4) , and  (6)  to take advantage of the computational efficiency in Matlab programming language. The Matlab code is presented in  Appendix A  that includes several functions in the service of the AMI computation.

In selecting Matlab functions against which to compare the algorithm presented above, we adopted the following criteria. The function’s code should be: (i) easily available either through Internet download or provided by the author(s), (ii) easy to implement for relative Matlab novices (see comments above), (iii) platform independent (Mac OS X, Windows, Unix, etc., a property inherited from Matlab itself), and (iv) a good representative of a broad class of algorithms of the type in use (e.g., histogram-based density estimator, kernel based with single bandwidth, etc.). We elaborate on the last criterion: our evaluation is meant to assess more general aspects of AMI estimation, not necessarily the specific code. That is, if a certain result obtains from the fixed class width histogram function we examine, it is likely to be true for all fixed class histogram-based algorithms so long as criterion (iv) is met—that is, this particular algorithm is a correct (error-free) version of that class. Four algorithms were selected according to these criteria ( Grinstead, 2006 ,  Peng, 2007 ,  Peng et al., 2005 ,  Qiu et al., 2009 ,  Stergiou, 2004 —obtained circa,  Stergiou, Kochi, Deffeyes, & Siu, 2008 ). We do not claim that these choices exhaust the available options, as any quick search on the Internet for code to compute AMI might reveal. These options sample the wide array of computational perspectives readily available to experimental psychologists and are representative of broad approaches to estimation.

Histogram-based algorithms . Three of the algorithms that were tested are best categorized as histogram-based in that they estimate the probability densities using binned-based histograms. Generally, these algorithms did not permit the user to input bandwidth choice. The simplest is an algorithm by  Grinstead (2006)  available from the  Mathworks.com  file exchange area (see references). The bin width for the histogram classes are selected by an adaptive algorithm that produces a partition of the data to roughly balance the proportions in the different classes (see  Neter, Wasserman, & Whitmore, 1992  for a discussion of the value of this type of class partitioning in the construction of histograms). The Grinstead algorithm uses Eq.  (2)  to estimate mutual information. The second histogram-based algorithm was obtained following a colleague’s attendance of an annual nonlinear time series analysis workshop ( Stergiou et al., 2008 ). The Stergiou function was designed for the specific application of time-delay embedding (see section below), but the argument input was modified for the current examination to accept any pair of variables not just a single time series. Nothing in the computational code was changed. It is a straightforward histogram-based density estimator with equal class intervals (not adaptive) that uses Eq.  (2)  to compute AMI. There is a fixed number of bins (128) that could be recoded if the user desired, but were not for this examination. Bin widths are determined automatically using the range of the data relative to number of bins. The third histogram-based algorithm is somewhat less user friendly in the sense of accessibility of computation (e.g., calls to compiled C++ code). It is due to  Peng  ( 2007 , which references  Peng et al. ,  2005 ). 
                            2 
                         
                         
                            2 
                            
                               Peng et al. (2005)  actually described kernel density estimation analogous to  Moon et al. (1995) . However, careful examination of the code available on the file exchange site at  Mathworks.com  reveals it to be a rather sophisticated histogram-based algorithm that appears to be somewhat adaptive. 
                          The bin boundaries for the histogram are adaptive in a manner similar to those introduced by  Fraser and Swinney (1986) . Though the programming code is quite a bit more complex than Grinstead, this algorithm also uses Eq.  (2) .

Kernel estimator algorithm . The fourth algorithm, due to  Qiu et al. (2009)  is a kernel density estimator coupled with Eq.  (3)  to estimate AMI. The kernel is a Gaussian that is standard normal in both the univariate and bivariate cases (i.e., has the identity as the covariance matrix in the latter). Hence, there is only one bandwidth used that is the same on all dimensions when estimating a bivariate density. It is described in detail in Qui et al., and is available from those authors.

In this section, we test whether the algorithms used to estimate AMI from obtained data are affected by the marginal distributions when they should not be. It has been shown that mutual information is strictly a measure of the nature and amount of dependence between a pair of random variables (e.g.,  Ma & Sun, 2008 ;  Mercier et al., 2006 ) and is not theoretically impacted by any aspect of the marginal densities themselves. In probability theory, it is possible to separate dependency between two random variables from characteristics of their marginal distributions. The device to do this is called a  copula function  ( Nelsen, 2006 ).  Appendix B  reviews mathematical details of copula modeling and its relevance to this validation test. For our purpose, copulas enable the simulation of any form of dependency between variables for any pair of marginal distributions. We use a “Gaussian” copula function

One of the most common applications of AMI is in state-space reconstruction of measured output of a nonlinear dynamical system ( Abarbanel, 1996 ,  Fraser & Swinney, 1986  and  Kantz & Schreiber, 2004 ). This type of analysis is central to psychologists studying the perception–action systems and motor control ( Stergiou & Decker, 2011 ) or clinical neurologists who examine EEG for signs of pathology ( Wackermann & Allefeld, 2009 ). In nonlinear dynamics, a (theoretical) phase–space is a diagram of the state of a modeled system that is usually constructed by plotting higher derivatives (such as velocity,

How many derivatives (i.e., dimensions) one must use to represent a system depends on its complexity. Damped oscillators only require two, whereas Lorenz systems require three. In the realm of measured data, one usually only has a single digitally measured time-series on hand for investigating system properties. Fortunately, due to a theorem of  Takens (1981) , the topology of a state-space is preserved if one ‘estimates it’ by reconstruction using the time-series data only. The process, often referred to as time-delay reconstruction or embedding, is detailed in classic texts such as  Abarbanel (1996)  and  Kantz and Schreiber (2004) . Briefly, one shifts down a few time steps into the series, copies all the data at that time step and beyond, places this copied series in dimension slot two, checks if it is necessary to retain the new dimension using nearest neighbor techniques, and repeats until no more new dimensions are required. The first pragmatic question in this process is, ‘how many time steps are necessary to shift down before copying the series?’ Takens’ theorem says any

In this section, we briefly illustrate one possible use of average mutual information that may be of interest to readers, though other applications to the analysis of behavior have been cited throughout. A growing area of research in neuroscience centers on connectivity analysis that can be used to reveal networks of neurons that subserve specific cognitive functions ( Haufe, Nikulin, Muller, & Nolte, 2013; Sporns, 2011 ). To delve more fully into this literature is far beyond the scope of this paper; suffice it to say that metrics purported to measure dependence between neural signals such as spike trains of individually recorded neurons or EEG channel signals are the principle means by which connectivity is assessed in these paradigms ( Ward & Doesburg, 2009 ). The idea is that if signals  
                         X 
                       and  
                         Y 
                       participate in a coherent network of neuron ensembles, they are likely to be statistically dependent or ‘synchronous’ during that epoch of time in which the network is active ( Singer, 1999 ). Average mutual information has been suggested as one such metric for situations in which linearity or stationarity of the signal cannot or should not be assumed (e.g.,  Quiroga, 2009 ).

The present illustration uses data obtained from one observer performing a visual odd-ball task in which the participant is to indicate by a button-press whether one of two stimuli, called the ‘target’, is present and to withhold their response when it is not. The target is a low probability event, occurring, with 20% probability within the block, while the ‘nontarget’ occurs with 80% probability.

                       This is a well-studied paradigm, with many variants (e.g.,   Luck et al., 2009 ) whose primary purpose is to identify electrophysiological activity associated with categorizing the novel or low frequency event relative to the high frequency event. The typical finding is an increase of the positivity of the averaged EEG waveform in the neighborhood of 300 ms post onset of the target relative to that of a nontarget trial. This is a very robust effect and is thought to indicate engagement of a frontal–parietal attention network whose purpose it is to direct resources to the encoding of the novel or important event ( Polich, 2012 ). We do not wish to add any new insights into this well-studied phenomenon here; our goal is to simply illustrate how one can use average mutual information to assess channel synchrony (in a broad sense) using EEG data from epochs of this task for two example illustrations: (i) channel data from sensors close to each other on the scalp should evidence higher AMI than pairs of channels who are further apart on the scalp, and (ii) the pair of electrodes

The primary purpose of this paper is to demonstrate the accuracy and utility of an algorithm programmed in the Matlab language for the computation of AMI between two sets of observations that is easy to use and transparent in its method of computation. Our EffAMI algorithm incorporates best practices in density estimation as well as efficient programming techniques to quicken the speed of computation. Specifically, we use kernel density estimation together with tuning of the kernel’s covariance matrix using attributes of the data (rather than using a circular density as is commonly done). Three validation exercises provided evidence on accuracy, bias, consistency, and sensitivity to irrelevant features of the data. The first of these measured accuracy, bias, and consistency of our algorithm and a set of representative alternatives for recovery of dependence in the simple case of a bivariate Gaussian random variable. In the second evaluation, we examined the extent to which skew on the margins contaminated the accuracy of AMI computation. Finally, because AMI is often used in the determination of an appropriate time-lag for phase space embedding, we assessed the geometric accuracy of embeddings resulting from time-lags suggested by each algorithm. From these observations, we make the following general conclusions: (i) kernel density estimation appears to be superior to histogram estimation of the probabilities needed to compute AMI and fixed class histograms perform poorly, (ii) using the data to ‘tune’ the bivariate kernel’s correlation parameter is better than using a circular kernel, (iii) within estimator type, bandwidth choice appears less impactful on accuracy though the normal reference rule works best when data are symmetric. Over all three evaluations, the algorithm we offer produced excellent results. Users of this or a similar algorithm to compute AMI on their data can feel reassured that the results are accurate, consistent, resistant to skew in the marginal distributions, and can be incorporated as an effective step in applications involving constructing phase embeddings.

We note, as we have earlier, it is likely that any algorithm that embodies the attributes we adopted will likely perform as well. One limitation of our function is the demands on available computer memory which is the direct consequence of the manner in which the summations are computed. The efficient aspect of our algorithm is brought about from the use of matrices required to compute double summations instead of the slower process of computing for loops (a process referred to as ‘vectorizing’ the computation). However, we do not believe this will continue to be a limitation for long, as hardwire development is rapidly rendering memory issues a problem of the past.
