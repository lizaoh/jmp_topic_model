So far, only weak utility functions have been used to represent strict partial orders, but such a representation is not completely satisfactory, since we can not reconstruct the preference relation by simply knowing the weak utility function. The purpose of this paper is to present a type of numerical representation for strict partial orders, in such a way that recovering the preference relation is possible. Taking into account that (2[R] , # ) is a strict partial order, we consider set-valued utilities, namely, set-valued mappings, such that they admit a selection which is a weak utility function. � 1999 Academic Press Key Words: numerical representation; strict partial orders; embeddings; set-valued utility. 

1. INTRODUCTION 

In social sciences, and especially in economics, binary relations appear in a natural way associated to either individual or social decision making. Consider a (finite or infinite) set of alternatives, X, and an agent having preferences over the elements in X. Those preferences are usually represented by means of a binary relation P, in such a way that, for any two elements x, y # X, xPy if and only if the agent strictly prefers x to y. For any x # X, denote P(x)=[ y # X | yPx]. A minimal requirement on P is asymmetry (for any x # X, y # P(x) O x � P( y)). An asymmetric binary relation is called a preference relation.

Preference relations satisfying transitivity and negative transitivity are called weak orders. Negative transitivity is a fairly strong requirement. It has been criticized for a number of reasons, such as, for instance, due to the problem of just perceptible differences (Luce, 1956) or changing or adapting tastes [see Elster (1982)] in the case of individual consumption, or to the intrinsic problem of aggregation of individual preferences for social decision making. 

A less demanding rationality condition consists of asking the preference relation to only satisfy transitivity. A transitive preference relation is called a strict partial order. Strict partial orders appear in a natural way, at both the individual and the social level. Suppose there is a consumer for whom commodities are desirable according to multiple attributes, in such a way that he is only able to compare when one commodity is better in all of attributes than the other, but he is unable to compare otherwise. Then, if X is his consumption set, his preferences over bundles behave typically as a strict partial order. Consider now a society made up of a set of individuals N, such that, for every i # N, individual i's preferences are weak orders, represented by utility functions fi , i # N. Take, for the society, the Pareto relation xPy if and only if fi (x)� fi ( y) for all i, with at least one strict inequality: all individuals like x as much as y, and at least one individual likes x better. Again we are in a strict partial order. As another example, consider the case of a consumer having preferences for flexibility [see Kreps (1979)]. Then, we may consider elements in X as potential sets of alternatives, in such a way that the larger the opportunity set, the better. Then, xPy if and only if x#y.[2] Again, these types of preferences (as the strict set inclusion) are strict partial orders. Consider also a nonutilitarian valuation of opportunity sets, but with a valuation in terms of freedom of choice [see Sen (1988), Pattanaik 6 Xu (1990)]. Again, a larger opportunity set is better from this point of view. These last two examples rely on the strict set inclusion being a strict partial ordering. Taking this fact into account, when looking for a numerical representation for strict partial orders, we shall consider the strict set inclusion on the subsets of the real line as our reference for a strict partial order. 

Generally speaking, a strict partial order cannot be represented by utility functions. Nonetheless, a less restrictive representation has been associated with strict partial orders. Suppose that a real valued function f: X � R can be defined in such a way that xPy implies f (x)> f ( y) or, in other words, P(x)�[y # X | f ( y)> f (x)]. Such a type of function, called a weak utility function, can be consistent with a strict partial order P, but, simply by knowing f, we cannot reconstruct the strict partial order it ``represents'' [cf. Fishburn (1970), Peleg (1970), Richter (1971), Majumdar 6 Sen (1976), Peris 6 Subiza (1995, 1997)]. Interval orders and semiorders constitute intermediate cases between weak orders and strict partial orders. Several authors have studied the problem of associating real-valued functions to interval orders or semiorders [see Fishburn (1973), Doignon, Ducamp, 6 Falmagne (1984), Bridges (1983, 1986), Chateauneuf (1987), Bosi 6 Isler (1995)]. Again, strict partial orders do not admit this pairwise type of representation. 

The aim of this paper is to provide a type of numerical representation of preferences suitable for strict partial orders, in such a way that we can characterize every P(x) by means of such a numerical representation. In so doing, we explore the possibility of combining weak utility functions with set-valued mappings on 2[R] endowed with the set inclusion, #. We call this type of representation set-valued utility . Section 2 introduces the concept of set-valued utility and provides some results of existence for strict partial orders, in the case in which X is countable or when P is separable. Section 3 analyzes continuity properties for the aforementioned representation. 

2. SET-VALUED UTILITIES 

Let us consider pairs (X, P), where X is a (finite or infinite) set of alternatives and P is a preference relation on X. Let us also assume a reference pair (Y, Q), where Q is a preference relation on Y. Consider now the usual concept of embedding: 

Definition 2.1. An embedding from (X, P) in (Y, Q) is a function f: X � Y such that y # P(x) if and only if f ( y) # Q[ f (x)]. 

A weaker requirement would be the following: 

Definition 2.2. A quasi-embedding from (X, P) in (Y, Q) is a function f: X � Y such that y # P(x) implies f ( y) # Q[ f (x)]. 

Utility functions and weak utility functions are particular cases of embeddings and quasi-embeddings, respectively, where the final pair is (R, >). In the same way as (R, >) appears in a natural way as the reference pair for weak orders, we will consider (2[R] , # ) as the natural reference pair for strict partial orders. Our intention is to work simultaneously with both (R, >) and (2[R] , # ). Consequently, ensuring the existence of maxima of f is a way of guaranteeing the existence of maximal elements of the strict partial order P. In the case X is endowed with a topological structure, this can be easily done by combining continuity conditions on f and boundedness conditions on X. 

In order to ensure the existence of continuous numerical representations for preference relations on topological spaces, two different tools have been used in the literature. One of them consists of requiring additional conditions on the preference relation. The other one consists of requiring additional conditions on the set of alternatives. 

Following the first approach, Peleg (1970) analyzes the existence of a continuous weak utility function for a continuous strict partial order on a topological space. Let X be a topological space and let P be a strict partial order on X. For x # X, denote L(x)=[ y # X | xPy]. The strict partial order P is continuous if L(x) is open for all x # X. Theorem 3.1. (Peleg 1970), If P is a continuous, separable, and spacious strict partial order on a topological space X, then there exists a continuous weak utility function for (X, P).
