The first one differs only in some minor aspects from a previous  result of Krantz (1968). The second one is new and is applicable in extensive measurement when the empirical set contains a  maximal and a minimal (nonnull) element. The generalization to the case of a minimal element is realistic in a number of  psychological applications (cf. the notion of a sensory threshold in psychophysics), and it is also interesting because it  necessitates the formulation of procedures for assigning numbers to objects that differ from those commonly used. The  application of these results in extensive measurement is briefly considered.  1. INTRODUCTION  Nlany results in measurement theory depend basically upon H61der's theorem, that any Archimedean fully ordered group is  isomorphic to a subgroup of the additive group of the real numbers (H61der, 1910; Birkhoff, 1967). Typically, one is given  some empirical set of objects, on which a concatenation and an order relation have been defined experimentally. Granted that  this empirical system satisfies a number of assumptions, an application of Holder's theorem yields a mapping of the set of  objects into the reals that is order preserving and additive with respect to the concatenation. Straightforward applications  of this theorem are not always possible, however, since empirical sets and relations are seldom well-behaved: the set may be  bounded, the concatenation may not be closed, the order relation may not be connected, and so on. Accordingly, a number of  generalizations of H61der's theorem have been worked out to permit analogous constructions of measurement scales under weaker,  more realistic assumptions (see for example Krantz, 1968; Roberts and Luce, 1968; Luce and Marley, 1969).  The present paper contains two generalizations of Holder's theorem. The first generalization (Sec. What is meant here by a nonnull element is an element x which,  concatenated with any other element _v, yields an element not equivalent to y. The assumption that the empirical set contains  a minimal nonnul element is interesting to consider for two reasons.  1. It contradicts a basic assumption of actual procedures for assigning numbers to objects. Indeed, suppose that some object x  is to be measured. Typically, some object y much smaller than x is chosen, and a "standard sequence" ly, 2y,..., ny,... is  formed (where ly is some object equivalent to y, and for n > 1, (n + I) y is some object equivalent to ny concatenated with  y.) Let N(y, x) be the largest integer n such that ny is smaller than x. A fixed element X, -a standard-is also taken. Then  the ratio N(y, #N(y, xl) is the measure of x. This procedure is justified by the axioms of extensive measurement, which imply  that: (i) N(y, x), N(y, xl) exist; (ii) the ratio N(y, x)/N(y, xl) is approximately additive with respect to the concatenation  of x; and (iii) this ratio converges when y is taken arbitrarily small. Hence, x xan be measured with any desired accuracy by  taking y sufficiently small. Clearly, the existence of arbitrarily small objects is essential to this method, and the question  is raised whether a measurement procedure can be divised which gives arbitrarily fine accuracy, even when this assumption is  not fulfilled. The answer is yes. Such a method is described in detail in Sec. IV.  2. The assumption that a minimal (nonnull) element exists in the empirical set is not unreasonable in a number of  psychological applications, in connection, for example, with the notion of a sensory threshold. In a psychophysical  experiment, suppose that a subject is asked to order a set of visual stimuli in terms of their subjective intensity (forget  about the variability for the sake of this example). These stimuli are generated by illuminating a rectangular screen with a  source of white light of varying intensity. The concatenation of two stimuli is obtained by illuminating the screen with two  different sources with the corresponding intensities. In such a case, a number of psychophysicists would certainly expect to  have a threshold stimulus. If only stimuli equal to or above the threshold are considered, this threshold stimulus plays the  role of our minimal element. This argument suggests that the structure of the  empirical set, and hence of the numerical set, is worth careful investigation.  It will be considered with a special attention in this paper. When Holder's theorem can be applied, the numerical set is the  positive subset of some subgroup of the additive real numbers. In the generalizations of Holder's theorem contained in Sees.  II-III, the numerical set can be some positive, bounded subset of the additive real numbers. Not every such subset is to be  considered, however. Intuitively, we would like the set of objects to be in some sense complete.  The corresponding notion for the numerical set is that of an "interval of a subgroup of the additive real numbers." This idea  is made precise in Definition 1 below in which all the basic notions of this paper are introduced, in particular, that of an  S-system. In some instances, the numerical set will be bounded above. In such cases, there may be two numbers, x, y say,  assigned to some objects, while x + y is not a member of the numerical set. Correspondingly for the empirical set, it cannot  be assumed in general that the concatenation is defined for every pair of objects. There will be a special set of pairs of  objects, for which the concatenation is defined. The notion of an S-system is simply a formalization of this idea.  DEFINITION 1.
