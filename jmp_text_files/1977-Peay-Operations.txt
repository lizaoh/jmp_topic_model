JOURNAL OF MATHEMATICAL PSYCHOLOGY 15, 89-101 (1977)  Matrix Operations and the Properties of Networks and Directed Graphs*  EDMUND R. PEAY  The Flanders University of South Australia, Bedford Park, South Australia, Australia  Matrix multiplication is frequently used to derive properties and quantitative features of directed graphs and networks from  their adjacency or value matrices. In many cases, a "modified" matrix multiplication is employed, where other binary  operations replace ordinary addition and multiplication. In this paper, a set of conditions for permissible value domains,  "addition" and "multiplication" operations, and value matrices is developed under which generalized matrix multiplication  directly yields properties of a digraph or network. An additional condition ensures that the properties are related to paths  of the structure rather than sequences in general. Finally, a number of applications of generalized matrix multiplication are  discussed.  Empirical structures in many areas of psychology are represented in terms of directed graphs and networks, where each point in  a directed graph, or digraph, corresponds to an empirical element, and directed lines joining the pairs of points represent  empirical relationships. A network assigns to each line of a digraph a value (usually an integer or real number, but  potentially another sort of entity such as a sign, a set, or a matrix). One of the most attractive features of this mode of  representation is that many of the properties' of the empirical structure may be conveniently derived from matrices associated  with the digraph or network, the adjacency matrix, and (for a network) the value matrix. (For an extended discussion of  digraphs and networks, see Harary, Norman, & Cartwright, 1965.) The matrix operation usually applied to an adjacency or value  matrix in order to derive some property of the structure is ordinary matrix multiplication. It is used, for instance, to  obtain the n-step transition probabilities for a Markov chain and the reachability matrix for an ordinary graph or digraph.  Often, however, the use of a "modified" matrix multiplication is required, in which ordinary multiplication and addition are  replaced by other binary operations. For instance, one may wish to find the value of a "cost geodesic" joining a pair of  points in a network; that is, a path whose cost (the sum of the values of its lines) is a minimum. Generalized matrix multiplication determined by different specifications for the "multiplication" and "addition" operations  has appeared in a number of settings (Harary, Norman, & Cartwright, 1965; Doreian, 1974; Peay, 1970, 1975), and similar ideas  can be seen in the techniques of fields such as mathematical programming (Hu, 1969). Cartwright and Gleason (1966) present an  involved, but practical (using the computer), algorithm for finding the values of sets of paths and cycles under a wide range  of operations for generalized "addition" and "multiplication." Robert and Ferland (1968) develop an algorithm for results of  generalized matrix multiplication in "Q-semirings" (see footnote 4). It is of both practical and theoretical interest,  however, to identify the class of properties of digraphs and networks which can be found directly from the "powers" of the  value or adjacency matrix derived using generalized forms of matrix multiplication.  The purpose of the present paper is to establish the value domains, "addition" and "multiplication" operations, and additional  conditions under which generalized matrix multiplication yields properties related to sequences in a digraph or network.  Further conditions are specified under which the properties related to paths (usually of greater interest than sequences) are  found. After some additional results, a number of examples are presented, some of which are familiar (but often approached by  more tedious methods), others of which do not seem to have been previously proposed but appear to be potentially useful.  PRELIMINARY DEFINITIONS  A directed graph (or digraph) consists of a set of points V together with a subset X of V X V. Each ordered pair (vi , v;) in  X is called a line, joining vi to v; . While the definition of the digraph usually excludes reflexive lines, i.e., lines of  the form (vi , v;), it is convenient here, for the present, to permit them. The ijth entry of the adjacency matrix is 1 if (vi  , v;) E X and 0 otherwise.  A network N consists of a directed graph together with the assignment of a value, an element from some set D (referred to as  the value domain for N), to each oits lines. The ijth element of the value matrix M = M(N) for N gives the value assigned to  the line (vi, v;). Only networks will be referred to in the following development, since a digraph can be treated in this  context as a network with a particular value matrix (to be discussed below).  A sequence joining vi to v; is a sequence of points (vi ,..., v;) in N such that a line in N joins each point in the sequence  to the next, together with this set of lines. A sequence of length n (or n-sequence) is a sequence having n lines (or n + 1  points).  An n-path is an n-sequence in which no point appears more than once.  Typically, the value of an n-sequence is defined by the product of the values of its lines, and the value of a set of  n-sequences by the sum of the values of its component n-sequences. For instance, ordinary matrix multiplication applied to the  value matrix  MATRIX OPERATIONS 91  for a network N gives, for each n, the value of the set of all n-sequences joining each pair of points of N (e.g.,  higher-order transition matrices). Other "values" for sets of nsequences can be defined by substituting different operations  for addition and multiplication (see examples given in a later section).  If certain conditions on D, M, and the "addition" and "multiplication" operations are met, then Mn, obtained using the  generalized matrix multiplication defined by these operations, gives the correspondingly defined "value" for the set of  n-sequences joining each pair of points. Furthermore, certain additional conditions imply that the value for the set of  n-paths is obtained. These conditions are the subject of the next section. VALUE MATRICES WITH ARBITRARY DIAGONAL ELEMENTS  The practical significance of Theorems 2 and 3 is extended considerably if the results are not limited to networks with value  matrices having 1L in the diagonal. In fact, the methodology implied by the foregoing theorems may be applied to networks  whose value matrices do not satisfy C8; i.e., some diagonal elements may not be equal to 1L . One very important example is  the irreflexive network or digraph, which contains no line joining a point to itself, and the value or adjacency matrix for  which has all diagonal elements equal to 0. The digraph, in fact, is generally defined to be irreflexive. Fortunately,  however, the principal results established above can be shown to be usable for such digraphs and networks. If Theorem 3 is true for  N', it is true for N, since the n-paths of N', and their values, are identical to the n-paths of N. Thus, M' can be used to  find the values of the sets of n-paths of N. Networks with Other Value Domains  While networks in empirical applications have generally had real values, the conditions presented above permit many other  kinds of value domains. Several examples are presented here to suggest the wide range of possible realizations.  1. The simplest nontrivial system which satisfies CI-CIO is the two-element set D = {0, 11, 0 = 0, 1 = 1, under Boolean  addition and multiplication. While several of the examples in the previous section provide a means for finding the  n-reachability matrices for a digraph, a value assignment using this domain and these operations gives a straightforward  2. The mathematical system for Abelson and Rosenberg's"Symbolic Psycho-Logic" (1958) can be seen as an application of a  particular value domain to a cognitive network under addition and multiplication operations which permit one to test whether  or not a given structure is balanced. Specifically, four relations are postulated: p (positive), n (negative), o (null), and a  (ambivalent), and the addition and multiplication tables show that this value domain satisfies C1-C9. Therefore generalized  matrix multiplication can be used, and it provides a means for determining whether a given structure is balanced which is  simpler than the one proposed by Abelson and Rosenberg.  Harary et al. (1965), using this form of generalized matrix multiplication, show that a structure is N-balanced (i.e.,  balanced when only semicycles of length N or less are considered) if and only if Mk, 1 < k < N, contains no diagonal entry a  or n. A simple matrix criterion for balance then follows.  3. Let D be g(R), the power set for some set R. This value domain, with an  MATRIX OPERATIONS 101  addition operation U and a multiplication operation r1, with b = o and 1 = R, satisfies Cl-CIO (and thus any distributive  lattice is also a value domain satisfying Cl-C10). Therefore, if the values assigned to the lines of a network are subsets of  some set R, generalized matrix multiplication may be used to obtain the (set-defined) values of sets of paths in the structure.  One interesting illustration of this is suggested by an example given by Harary et al. (1965). Let the points of a given  network represent the members of an organization and each line indicate that one member is related to another in some way. If  R is the set of possible relationships, such as "liking," "authority," or "communication," then the value assigned to each  line is a subset of R consisting of the set of relationships joining the two given individuals.
