Working memory,  Baddeley (1986) , is a transient form of memory that maintains task relevant information during conditions of competing demands. As such, it is thought to be involved in all forms of cognition and is often one of the first types of memories to show impairments associated with aging. Understanding this form of memory is of critical importance to neuroscience. In humans, working memory capacity is typically measured with tasks in which subjects are asked to retain a certain number of distinct objects in memory (e.g., words or visual symbols). From these tasks the working memory capacity in humans has been estimated to be between 4 and 7 objects depending on the task used ( Cowan, 2000; Miller, 1956 ). However, measuring working memory capacity is much more difficult in non-verbal laboratory animals. The most common task employs the radial arm maze, which has been used to assess working memory in a vast gamut of laboratory animals including bees ( Brown & Demas, 1994 ); rats ( Cole & Chappell-Stephenson, 2003; Olton & Samuelson, 1976 ); pigeons ( Roberts & Van Veldhuizen, 1985; Spetch & Edwards, 1986 ); gerbils ( Wilkie & Slobin, 1983 ), gorillas and marmosets ( MacDonald, 1994; MacDonald, Pang, & Gibeault, 1994 ). However, despite this extensive use of the radial arm maze, there is no mathematical model that allows one to estimate memory capacity from performance data on the maze.

In the radial arm maze task, originally developed by  Olton and Samuelson (1976) , an animal is placed in the hub of a maze with a fixed number of arms radiating outward from the center (typically eight arms). At the start of the task all the arms are baited with food and the animals are free to traverse the maze and consume the food. Initially, the animals have no knowledge of the contingencies associated with the maze and as such they wander the maze in a seemingly random fashion, visiting some arms multiple times. Over subsequent trials, they are thought to form an internal representation of the spatial layout of the maze and use this knowledge to navigate the maze efficiently (minimizing the number of repeat visits). However, to do this, the animals must also rely on working memory to retain a record of which arms they have already visited during a trial. Performance on the task is commonly evaluated in the radial arm maze literature using one of two different measures, the mean number of errors (repeat visits) accumulated once all arms have been visited or the mean number of correct choices (a visit without a repeat) in the first  
                         m 
                       choices, where  
                         m 
                       is the number of arms. How these performance measures relate to estimates of (spatial) working memory capacity has not been quantified.

In this paper, we present a mathematical model for the radial arm maze task that relates memory capacity to each of these performance measures. We model performance in the radial arm maze via random search with memory. In the simple model presented here, the animals are thought to rely on working memory to maintain a memory trace of a fixed number of previously visited arms. They avoid visiting the arms maintained in memory but otherwise they behave randomly. This complex behavior is in some ways analogous to the behavior of simple organisms such as bacteria which follow concentration gradients of food by biasing their random movement towards areas with more food. For a description of this phenomenon, see  Staddon (1983) .

In our model, performance depends on three factors: the number of arms in the maze, the fixed memory capacity, and the search strategy used to navigate the maze. (e.g. investigate new arms uniformly randomly or visit new arms using some other random search strategy, for example, turn clockwise to the next immediate arm 70% of the time and to the next following arm 30% of the time). In the case of a uniform random search strategy, we derive explicit expressions for the density, the distribution, the mean and the second moment for the errors accumulated when all arms have been visited. We derive an algorithm to compute the probability density of the number of correct choices in the first  
                         m 
                       choices, and from the algorithm, we derive an explicit expression for the density. For the case in which the search strategy is not uniformly random, we use Monte-Carlo simulation to estimate the probability density for each of the performance measures as a function of a specified search strategy, number of arms in the maze and the memory capacity.

To put our model in context, we note that  Dallal and Meck (1990)  define a continuum of solution strategies for navigational tasks, such as the radial-arm maze task. They begin with the most basic strategy, random, unsystematic sampling of locations requiring no reliance on memory. Next, they specify algorithms, response strategies that determine the next location to be visited, which are thought to make minimal demands on memory. They then specify spatial mapping, requiring both reference and working memory to encode information about a subject’s spatial relationship to its environment. Finally, they list chunking strategies which rely on hierarchically structured working memory to provide increased information processing. Our model presents another navigation strategy, random search with finite memory. We address search strategies by simulation. These strategies are specified by sampling from non-uniform transition probability densities on the arms.

We define the model in Section  2 , and describe its implementation. In Section  3 , we present derivations of the closed form expressions for the probability densities, and give an example of simulation of non-uniform random search. In Section  4 , we compare model predictions to experimental results. We begin with an illustration of the basic model (uniform search), then address non-uniform search and its implications, and finally consider an extension of our model to a hierarchical maze and to an example in which some arms are left unbaited. Finally, in Section  5 , we discuss the implications and limitations of the model.

We specify a model for memory, the choice process (accept or reject a sampled arm), and the arm to arm search strategy. We model memory, MQ, as a queue with finite capacity  
                         c 
                      . Our results do not depend on the exit protocol for the queue as long as when the queue has reached its capacity and a new entry is inserted, exactly one old member of the queue exits. For example, a first-in, first-out (FIFO) exit protocol is consistent with the recency without primacy noted by  Olton and Samuelson (1976) . On the other hand, a last-in, first out (LIFO) protocol is consistent with primacy.  Kesner and Novak (1982)  show that, in some tests, rats exhibit both recency and primacy in short-term memory, so some flexibility in exit protocol is warranted. We model search strategies as random samples from a fixed density. The initial sample arm is (uniformly) randomly selected. The choice process proceeds as follows. Given an arm

We emphasize the difference between the next arm to be investigated (determined from the search/sampling process) and the choice process (accept or reject an arm). This is consistent with experimental observation, ( Brown & Cook, 1986; Olton & Samuelson, 1976 ). Animals tend to align along an arm and may partially travel down it, but only if they travel to the end of the arm and probe a raised platform are they considered to have chosen the arm, otherwise they are considered to have rejected the arm.

We assess the impact of non-uniform random search in the model by simulation. We illustrate this using two examples of highly non-uniform search. One is close to a deterministic strategy that is guaranteed to achieve perfect results on an 8-arm maze and the other is similar but will never complete the maze. Clockwise (or counter-clockwise) search is often observed in animals running a maze, see  Dubreuil et al. (2003)  for a discussion. In their paper on the radial arm maze,  Olton and Samuelson (1976)  tested rats in an 8-arm radial maze, and measured performance by the mean number of correct choices among the first 8 choices. They concluded that rats achieved, on average, 7.6 correct choices among the first eight with a range of 7.2–8.0. They proposed a memory capacity of 6. They also addressed chance performance ( 
                         c 
                         = 
                         0 
                       in our model), citing the occupancy problem ( Feller, 1957 ). We get similar results with our model at  
                         c 
                         = 
                         0 
                      . Assuming uniform random search, our model predicts a mean of 7.5 correct choices with a standard deviation of 0.5 at a memory capacity of 6. Thus our model is consistent with Olton and Samuelson’s estimate of capacity. They also observed that the rats “rapidly came to choose an average of more than seven different arms in the first eight choices” (page 99). In other words, after sufficient training, on average, more than the first 7 choices were correct, and so it is consistent with our model prediction that the first  
                         c 
                         + 
                         1 
                       choices are correct. Similar conclusions can be drawn with regard to the number of errors from  Dubreuil et al. (2003)  who investigate the performance of rats on an 8-arm maze comparing a protocol with confinement (subjects are confined in the hub of the maze after each choice for 5 seconds before being allowed to select the next arm) to a protocol without confinement. They show that the (asymptotic) performance of the subjects trained without confinement was comparable to that of subjects trained with confinement, although the confinement subjects took more trials to achieve the same level of performance. In both groups, the mean number of errors was near 1.0, and the first error occurred, on average, after 7 choices had been made. This is consistent with our model with memory capacity 6 which predicts a mean number of errors = 1.0.

Cole and Chappell-Stephenson (2003)  record data on the search strategies utilized by their rats averaged over trials and individuals, and the results are clearly non-uniform. However, because search strategies vary across individual animals, averaged search strategies may be misleading. In the next example, we simulate the impact of search strategies of individual animals on their performance.

Spetch and Edwards (1986)  use an open-field, 8-arm maze to study the spatial memory performance of pigeons. Next, we consider the case of a 16 or 24 arm hierarchical maze in which 2 or 3 secondary baited arms branch from the end of each of the 8 primary arms, see  Roberts (1979) . Roberts reports that rats trained on these mazes were able to visit all secondary arms with few errors (on average, 1.2 for the 16 arm case and 2.5 for the 24 arm case). He concludes that rats trained on this maze have a memory capacity in the range 24–32, organized as a spatial map, and suggests that the choices for primary and secondary arms are encoded in separate memory systems.

We have presented a model of performance on the radial arm maze. Our model adopts a finite queue to represent working memory and depends on two parameters, the number of arms in the maze and a finite memory capacity. In its simplest form (uniform random search), we have derived explicit expressions for the probabilities and moments for the number errors accumulated when all arms have been visited. We also derived an explicit expression for the probability of number of correct choices in the first  
                         m 
                       choices and for the rank of the first error. Simulation allows us to extend the model to non-uniform search strategies. Thus, we can quantitatively relate the number of arms in the maze, memory capacity and the two most common measures in the experimental literature.

We model memory as a queue of fixed capacity, and our proofs in Section  3  show that our results do not depend on any particular exit protocol, as long as when memory is full, exactly one current occupant of memory is ejected when one is added. Although the exit protocol does not affect the error statistics, it does have implications, for example, in the case of a hierarchical maze and the partially baited maze discussed above. On the other hand, a model such as fading memory in which items stored in memory exit in a random fashion with, for example, a probability depending on its location in the queue, will give different results from those presented here, as long as there is a non-zero probability that more than one occupant of memory exits before the next arm is selected.

Our model characterizes errors due to the finite nature of memory, but other internal sources of error are not treated. For example, we have assumed that the choice process (accept or reject an arm) is errorless. One type of error, falsely rejecting an arm that is not in memory, is relatively benign. If this error happens with a small probability, the incorrectly rejected arm will eventually be correctly accepted. On the other hand, accepting an arm that is in memory creates an extra error and, temporarily, effectively shortens the capacity of memory. The latter aspect of this problem could be eliminated by including a maintenance function in the model that eliminates copies during memory update.

Finally, deviations from random search strategies are largely taken into account by the fact that we model search as a sample from a discrete transition density, and as we have shown, even relatively large deviations do not significantly affect the results, except at low performance levels.

However, in all of our examples, the search depends on the relative and not the absolute location of the current arm. This is not likely to be the case in several types of non-symmetric maze arrangements, for example, mazes with arms of different widths ( Olton & Samuelson, 1976 ), unequal spacing of arms ( Cole & Chappell-Stephenson, 2003 ), or non-radial mazes ( Spetch & Edwards, 1986 ). Nevertheless, if we have data to support a transition density, our simulation can treat these cases.

Our model provides predictions that are consistent with the experimental literature, and we draw some novel conclusions about memory capacity. In the examples we treat, a modest working memory capacity in the range 3–9 is sufficient to explain the observed level of performance, even for large mazes. This contrasts with much larger estimates for memory found in the literature. Similarly, we extend the model to analyze the memory capacity required to explain performance on hierarchical mazes of the type studied by  Roberts (1979) , and once again a modest working memory capacity in the range 8–9 is sufficient. Finally, we note that it is common in the literature to compare experimental results to random behavior (i.e.
