A paradigm to  which many of these methods apply involves interpreting the data matrix as defining a binary relation R from the set A of  subjects to the set M of response items. Generally assumptions are made which permit the task of determining the minimum  number of dimensions mediating response with which the data are consistent to be recast into the problem of finding as small  as possible a family I' of relations of a particular type such that  n F=R (U I'= R). SUMMARY  A host of algebraic data represenation techniques are currently emerging (Bouchet, 1971; Cogis, 1980; Doignon et al., 1984;  Chubb, 1985; Doignon & Falmagne, 1984; Chubb & Peters, in preparation). The methods in this class generally seem to present  serious computational difficulties; in particular, for any data set (A, M, R), and any of the representation families 0 that  have been studied thus far, it is NP-hard to determine O^-dim R (O"-dim R) (see Definition 2.3).  Consequently, the practicability of a given algebraic represenation method depends in large part on the degree to which the  data set (A, M, R) to be represented can be collapsed prior to processing. We have defined a couple of distinguished  restrictions, then-core and U-core of any irreducible data set (A, M, R) such that [for convenience we refer to then-core;  similar remarks pertain to the U-core of (A, M, R)]  (i) it frequently turns out in practice that for n-core(A, M, R) = (a, p, C), ICI is significantly smaller than IRI,  (ii) n-core(A, M, R) may be constructed at polynomial expense,  (iii) for an important subclass 0 of n-representation families (including 0a, the family of all biorders), the task of  obtaining, for any OEO, a minimal 0^representation of (A, M, R) is polynomially reducible to the same task with respect to  n-core(A, M, R).
