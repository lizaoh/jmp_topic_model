Uniqueness of the obtained solution is proven. A solution algorithm is given for cases in which some of the testing times are zero. Some numerical examples are given.  1. INTRODUCTION  In previous work Horst (1949, 1956) and Taylor (1939, 1950) have provided solutions to the problem of maximizing the multiple correlation of a predictor set subject to variation in the relative lengths of the predictors for a fixed total testing time. The solution offered by Horst is simpler and more general than that offered by Taylor; however, each of these solutions can be guaranteed to be valid only if the true optimum lengths are strictly positive for all tests. If the true optimum solution involves one or more zero testing times, then the Taylor and Horst formulas will typically yield an incorrect solution suggesting negative testing times. The correct solution cannot then be obtained by assigning zero testing time to the tests having negative times in the Taylor or Horst solutions and by adjusting the positive times proportionally. Also neither Taylor nor Horst show that their obtained solutions are unique. In this paper we shall provide a more general solution to this problem, one which avoids all of these difficulties. The outline of this paper follows. We shall  1. obtain the extremum for variation in the regression weights with the testing times considered fixed;  2. show that this extremum is a unique maximum;  3. obtain the extremum for variation in the testing times with the regression weights considered fixed at their conditional optimum values, when the solution involves all positive testing times;  4. show that this extremum is a unique maximum;  5. show that a unique solution exists when some of the testing times are zero; 6. describe the basic properties of the solution as a function of total testing time; 7. provide a solution algorithm in the case in which one or more of the testing times are zero;  S. provide computational procedures and supplementary formulas; 9. provide some numerical examples with accompanying discussion.  The work in this paper should be of interest for at least four distinct reasons. The solution is certainly.of substantial practical value for the stated problem and to test constructors who desire to determine the optimum relative lengths for subtests of a test. Such applications are now being made at Educational Testing Service in a number of testing programs. Indeed, the methods developed here should also find application in scientific areas other than psychology in which error of measurement is an important factor. The matrix algebra methods used here (due primarily to the first author) should find wide application. When a maximization is required jointly with respect to two sets of variables it is often useful to fix one set, obtain a conditional maximization with respect to the second set, and then, with the second set fixed at these conditional optimum values to obtain the further maximization with respect to the first set.  The method of handling the boundary conditions and the development of the solution algorithm (due primarily to the second author) should also be of general interest. Certainly boundary conditions must be included explicitly in the formulation of maximization problems. A reviewer of this paper has suggested that this requirement "has seldom been met in previous work in psychometrics." The present paper describes just one way of handling such problems. Finally, it should be noted that the present problem is a continuous analogue of the classical, and still much discussed problem of selecting the best n from a set of N > n available predictors. Since no entirely satisfactory solution to that problem is yet available it may prove useful to attempt to imbed that problem within the framework of the continuous process considered here.  244 WOODBURY AND NOVICK  2. SOME PROPERTIES OF THE SOLUTION AS A FUNCTION OF TOTAL TESTING TIME  If the available testing time does not yield a solution with all positive testing times, an algorithm may be adopted which will always yield the required solution. In order to establish the correctness of this algorithm we first list some of the basic properties of the solution:  THEOREM 8.1. If ti > 0 when the total testing time is T, then ti > 0 when the total testing time is greater than T.  Proof. The proof of this theorem follows from Theorem 9.1.  THEOREM 8.2. If each variable has a nonzero true-score regression weight with criterion then, for T sufficiently large, all predictors will have nonzero testing times.  Proof. The proof follows on noting that the marginal increment in the multiple correlation decreases to zero as any predictor is lengthened. Hence, at some point, for any given predictor the marginal increments for all other predictors must be less than for it.  THEOREM 8.3. If for some time T and optimal allocation t*, e't* = T, all partial regression weights are positive, then for the optimal allocation for any time T', all partial regression weights are positive.  Proof. If for T' > T some Ni is negative while it is positive at T, then by the intermediate value theorem since Ri is continuous in T there exists a T", T' > T" > T, such that Ni = 0. But then ti = 0 at total length T", which is a contradiction. If T' < T and ti > 0 for T* = T but ti = 0 for T* = T', then the theorem still holds since the value of Ri in the second case is arbitrary.  VALIDITY OF A TEST BATTERY 251  THEOREM 8.4. If all true-score partial regression weights are positive, then for the optimal allocation t*, for total time T, e't* = T all partial regression weights are positive.  Proof. Suppose we have an optimal solution t* for total time T with all ,gi > 0, and with one or more negative observed-score regression weights, while all true-score regression weights are positive. As the ti are increased optimally the observed-score weights converge to the true-score weights. Hence those with negative weights initially must change sign. But the solution is continuous and hence, at some point, these weights must be zero. However, if this is so, that variable must then have a zero time allocation, which is a contradiction.  Some comment on the last two results is in order. These theorems do not amply that for any n-variable set having all positive true-score weights, that if some subset is chosen, all regression weights will still be positive for all relative lengths. This need only be true in those subsets -which are optimum subsets and only at the optimum lengths.  Finally we may state  THEOREM 8.5. If all true-score partial regression weights are positive, then for T* sufficiently large Eq. 12 will yield a vector t* of time allocations having all positive components. SUMMARY  We have shown how it is possible, in practice, to obtain the unique specification of the relative test lengths so as to maximize the multiple correlation of a predictor set with a fixed criterion. The steps are as follows: (i) Use Eqs. 42 and 43 to obtain the scaled covariance matrix F and scaled criterion covariance vector y. (ii) Obtain the scaled true-score partial regression weights, F-1y. (iii) If any variables have negative scaled true-score partial regression weights, reflect these variables and work with the new set of variables which will have all positive scaled true-score weights. (iv) Attempt to obtain a solution vector t for the relative times by using Eq. 12 with the available time T*. (v) If step 4 yields a vector having all nonnegative elements, then it constitutes a valid solution. If one or more of the elements are negative, then the smallest value T* which yields all nonnegative elements may be determined from (36). Under the very weak assumptions given in Sec. 1 a solution will always be possible to determine this value of T*. (vi) To obtain solution for smaller values of T* solve (12) for t( and determine the variables which first drop out of the equation as T* decreases. Denote this value by T,a . If T* has been determined from step 5 at least one variable drops out immediately. (vii) To obtain a solution for T* G T,,, obtain the new matrix F and vector y by eliminating the variable having zero time allocation at T = Tn . (viii) Use Eq. 12 to obtain solutions for T <_ T,, . The lowest value of T for which this solution will be valid can be determined by examining (36) to determine the time T,,-, at which the (n - 1)st variable receives zero time allocation. (ix) Repeat steps (vii) and (viii) to determine the solution for t in the successive intervals Ti G T < Ti_, , i = 1, 2,..., n. (x) For any solution vector t, with zeroes inserted in the appropriate places for all tests having zero time allocation, determine the corresponding regression weights and the squared multiple correlation from (45) and (49). (xi) If any variables were reflected in step (iii) now reflect the regression weights for these variables. The body of the paper is concerned with showing that a unique solution to the stated problem is always produced by this solution algorithm.
