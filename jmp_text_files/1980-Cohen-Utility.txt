INTRODUCTION  One common construction in mathematical psychology is that of a random utility system. For a finite set B called the choice  set, one is given a family of probabilities Qa,c called choice probabilities, indexed by pairs (a, C), where a E C and C C B.  These Q,,, are to be interpreted as the probability that element a has at least as much of some attl ibute as each element of  C. It is natural to inquire whether this system of probabilities has a representation of the form Qa,c = P{Up > Ub: for each b  E C), where {U": a c- B) is a set of random variables and P is the joint probability measure on this set.  Recently, Falmagne (1978) has presented necessary and sufficient conditions for the existence of such a random utility  representation. This paper extends his work in that the choice set B is allowed to be infinite but every choice probability is  indexed by ordered pairs (a, C), where a E C and C C B is finite. The interest of this extension is clarified by the following  Suppose one is given a finite set of multitone sequences of wide band white noise in which each noise burst is identified by  its intensity. One asks the subject which element of the sequence is the loudest. In this way one gets estimates of  probabilities P,.c: the probability that noise a is judged louder than each noise in the set C of noises. It is natural to fit  such data by a random utility model. However, the choice set in this case is an arbitrary finite set. It would be advantageous  to extend this model to the entire sensory continuum so that further predictions about different choice probabilities would  follow. The study of random utility representations on infinite choice sets serves to delimit the range of possible extensions. The finiteness of the empirically constructed choice set is due to necessary phenomenological constraints on the possible  outcomes of any experiment which are entirely independent of the theory of random utility representations.  Furthermore, one can often prove simpler and more elegant theorems about an infinite structure than about the corresponding  finite one. The infinite case might well serve as a model for the set of finite structures. Finally, the case of an infinite  number of choices has a certain amount of intrinsic interest independent of the finite case. Certain questions which have an  immediate answer in the case of a finite set of choice probabilities become worthy of considerations when the set of choice  probabilities becomes infinite. This paper revolves around the study of the interrelation of three of these questions.  Specifically, the countable additivity of the representing measure, the measurability of the representing functions, and the  relation of these properties to the cardinality of the choice set are studied. First we show that Falmagne's criterion for the  existence of a random utility representation is also necessary and sufficient for the existence of such a representation  defined on a choice set of arbitrary cardinality. However, we must give up the countable additivity of the resulting measure.  Second, if we give up the criterion that the representing functions are random variables (i.e., measurable), then we may  obtain a "random utility representation"- Falmagne's criterion still suffices. In the case that we require a countably  additive random utility representation we may still always obtain such when the choice set is countable, if Falmagne's  criterion is met. Moreover, we may choose our measure space to be a sub-v-field of Borel measureable sets on the real line.  However, if we require the representing functions to be random variables, and the representing measure to be countably  additive, and do not restrict the cardinality of the 'hoice set, then we obtain a random utility representation in a much more  restricted set of circumstances. A series of sufficient conditions are imposed which imply both the continuity of the  resulting measure with a naturally imposed Borel-LebBsgue measure and a countably additive random utility representation. One  of these results is noteworthy in that it is necessary and sufficient for a countably additive random utility representation  continuous with the Borel-Lebissgue measure discussed above. However, this condition is, unfortunately, only indirectly  expressed in terms of the given set of choice probabilities. A sufficient condition however, which is defined directly in  terms of the choice probabilities is discussed.  2. WEAK REPRESENTATIONS  2.1.
