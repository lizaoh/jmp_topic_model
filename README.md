# JMP Topic Models

To run using Binder, right-click on this badge and open in a new tab.

[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/lizaoh/jmp_topic_model/HEAD)

Double-click on the `notebooks` folder then open `jmp_topic_models.ipynb`.

Feel free to run the entire notebook using Run > Run All Cells then navigate to `Similarity network` in the table of contents (the 3rd button on the left side with the 3 bullet point lines).
