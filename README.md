# JMP Topic Models

To run using Binder, right-click on this badge and open in a new tab.

[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/lizaoh/jmp_topic_model/HEAD)

It'll open a JupyterHub. Please follow these steps:
1. Under "Other", open a Terminal.
2. In the Terminal, please enter the following command:

    `topicexplorer notebook jmp_text_files.ini`


3. Enter `n` to not overwrite the existing `corpus.py`. Wait until it finishes running and mentions that no web browser is found.
4. On the left side, double-click on the `notebooks` folder and open `jmp_topic_models.ipynb`.
5. You should be able to run the notebook :) Feel free to run the entire notebook using Run > Run All Cells then navigate to `Similarity network` in the table of contents (the 3rd button on the left side with the 3 bullet point lines).
